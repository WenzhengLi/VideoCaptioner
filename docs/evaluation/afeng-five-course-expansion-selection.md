# 阿峰方法层 5 课扩展集选择

> 日期：2026-07-16  
> 目标：在三课试验通过后，以最少新增课程覆盖新的事实层风险类型。

## 结论

5 课扩展按“总计 5 课”执行，不是额外增加 5 课。

固定集合：

```text
C003, C006, C007, C010, C012
```

其中 C003、C006、C010 已完成真实 MiMo 试验；新增 C007、C012。

## 选择原则

选择指标只用于衡量事实层和证据层复杂度，不使用旧 P05 安全风险数量作为阿峰方法层选择条件。

优先覆盖：

1. 多案例与案例边界变化；
2. OCR/课板信息密度；
3. speaker 不确定性；
4. P03 未分配片段比例；
5. 清晰基准案例；
6. 单案例过宽或过窄边界。

## 五课覆盖矩阵

| 课程 | 主要特征 | 选择用途 |
| --- | --- | --- |
| C003 | 3 个 v003 案例；历史 unassigned 约 40.41% | 多案例、边界变化和高未分配率回归 |
| C006 | OCR 681 次、课板 424；2 个案例 | OCR 密集、语音与课板融合；已出现人工复核案例 |
| C007 | unknown speaker 约 12.25%；课板 428；单案例 | 高 speaker 不确定与可能过宽案例边界 |
| C010 | 单案例；边界相对清晰 | 稳定基准和 Prompt 回归 |
| C012 | unassigned 约 54.03%；单案例 | 极高未分配率和可能过窄案例边界 |

## 为什么不优先选择其他课程

- C002/C011：多案例能力已由 C003 覆盖，可作为下一批扩展候选；
- C004/C009：长视频和较高 OCR 调用具有价值，但与 C006 的重型输入覆盖部分重叠；
- C008/C015：高未分配率有价值，但 C012 更极端；C015 可用于 v003 变更回归；
- C001/C005/C013/C014：整体指标更接近常规样本，适合 15 课扩展而非首个压力测试集。

## 验收指标

5 课完成后汇总：

- 案例总数；
- published / manual_review / rejected / failed；
- 首轮通过率和平均修订轮数；
- evidence ID 合法率；
- 必需 evidence 覆盖率；
- 单案例耗时和成本；
- speaker 不确定案例与高 unassigned 案例的忠实度差异；
- C006 人工复核问题是否在 C007/C012 重现。

只有 5 课没有出现系统性漏证据或错误发布，才进入 15 课扩展。
