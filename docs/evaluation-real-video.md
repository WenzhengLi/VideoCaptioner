## 兼容性结论

- PaddleOCR 3.7.0 在 Windows CPU 下搭配 `paddlepaddle==3.2.2` 可用；3.3.1 会触发 oneDNN PIR 属性异常；
- WeSpeaker 当前 UMAP 需要 `scikit-learn<1.8`；
- WinGet 安装的 FFmpeg 可自动发现，不再依赖当前进程 PATH 已刷新；
- CUDA 未验证。
