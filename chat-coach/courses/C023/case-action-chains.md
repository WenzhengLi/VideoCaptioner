# C023 案例动作链

## CASE-C023-001：女 DJ 爱恨纠葛

1. 女生展示独立和高门槛 → 讲师不立即对抗 → 先听她讲完整 → 找到孤独和被理解需求。
2. 她强调追求者多、见过世面 → 不加入竞争者队列 → 用理解与轻调侃拆掉人设压力。
3. 一次进攻没有接住 → 及时回收 → 换到她愿意谈的话题 → 情绪恢复后再拉近。
4. 情人节邀约与到家推进 → 她在行动上配合但口头仍设门槛 → 以行动信号校准，而非只听表面强硬。
5. 她到家后说要走 → 观察是否真的行动退出 → 给台阶而非纠缠 → 她留下并继续表达。
6. 她哭诉、索要拥抱 → 先接住情绪 → 再决定拉近；课程最后强调对她好一点，并限制学员照搬重手法。
