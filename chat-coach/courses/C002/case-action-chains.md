# C002 案例动作链

## 案例一：阿峰与高窗口女玩家

### 链路 A：主动招呼到“随缘”

```text
女生添加后主动招呼
→ 课程判断初始窗口存在
→ 男生表达没有爱情会枯萎、主动才有故事
→ 女生两次说随缘
→ 课程判断不是接受也不是拒绝
→ 不逼她表态，继续创造具体互动
```

### 链路 B：共同记忆到私密邀约

```text
聊到小浣熊干脆面
→ 男生提出买来一起吃
→ 地点进一步落到家里
→ 女生发省略号
→ 男生点出她突然变冷
→ 女生说第一次见面直接去家里很唐突
```

### 链路 C：理性顾虑转情绪画面

```text
女生承认喜欢，但认为直接到家太快
→ 男生先给外面见面的台阶
→ 感觉好再一起回去
→ 用“不想彼此喜欢还孤独回去”重建共同感受
→ 女生开始问见面地点并提出喝酒
```

### 链路 D：男生失联与女生高投入

```text
见面意向已出现
→ 男生长时间处理其他事情、不回复
→ 女生连续追问位置
→ 猜测有其他女生
→ 指责钓鱼、欺骗并大量发消息
→ 课程判断她投入和情绪都很高
```

### 链路 E：事实解释与态度

```text
女生认定男生撒谎
→ 课程要求提供详细时间线和事实
→ 男生明确“没有就是没有，我不会欺骗任何人”
→ 不为不存在的事情认错
→ 同时说明自己正在处理什么、何时能执行
```

### 链路 F：朋友框架与行动要求

```text
女生说做朋友也行
→ 课程不按字面理解为纯朋友
→ 判断她已厌倦空谈和反复逗弄
→ 要么真正见面继续，要么停止浪漫表达
→ 关系答案交给行动
```

### 链路 G：经期保险栓与外卖台阶

```text
女生说来例假
→ 课程判断包含测试、报复和保险栓
→ 男生不争论结果
→ 表达很多事不用一晚做完
→ 女生继续确认食物和家中条件
→ 提出叫外卖
→ 形成去家里的可执行步骤
```

### 链路 H：反复失约到胜负心

```text
女生再次出发或等待
→ 男生又因其他安排失联
→ 女生等待很久、删除又重加
→ 课程严厉批评连续放鸽子
→ 女生后期从喜欢叠加到“必须拿下”的胜负心
→ 最终见面不能反推前面的失约是正确动作
```

## 案例二与三：北京电商女生

### 链路 A：主动行动窗口

```text
女生持续发语音并主动约
→ 开车来接讲师
→ 一起吃海底捞
→ 课程判断行动配合远高于口头热情
```

### 链路 B：工作讨论与圈层证明

```text
双方讨论抖音电商
→ 对 IP 带货和货带人发生争论
→ 男生展示朋友圈中真实带货成果
→ 女生看到其圈层、视野和行业连接
→ 即使没有持续情感话题，仍形成同行感和继续相处意愿
```

### 链路 C：共同成长筛选

```text
讲师区分只消耗时间的相处
→ 与能提供新知识、资源和共同成长的人
→ 长期对象不只看外形和短期情绪
→ 还看双方能否互相带来成长与生活价值
```
