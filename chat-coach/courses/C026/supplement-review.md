## SUP-C026-001：聊天很浅但她一直在回

- 对象编号：`OB-C026-001`
- 场景：跨周低频短聊，几乎没有长对话，但没有断联。
- 女生原话：类似“躺着”“刚下班”“没有。”
- 讲师判断/处理：持续回复本身就是窗口；不因每句短就判死刑，也不强行聊深。
- 发现的缺口：缺少低信息场景的四种推进方式。
- 候选补足：A 状态接续（稳定回）：①“下班就先放空十分钟。”②“今天比昨天早一点。”③“躺着最适合选个轻松话题。” B 具体二选一：①“想吃东西还是直接睡？”②“休息是看电影还是出门？”③“明天早班还是晚班？” C 坏坏轻挑：①“你今天的聊天电量又只剩一格。”②“每次省字省得很稳定。”③“好，批准你用三个字回答。” D 等行动窗口：①“下班位置发我看看。”②“休息日确定了告诉我。”③“哪天想出来，直接给时间。”
- 后续分支：开始多说→沿题展开；仍短但持续→保持低压；给位置/时间→转行动；不回→停止连发。
- 补足理由：让“持续回复是窗口”不等于持续盘问。
- 来源边界：判断来自课程；句子为候选。
- 确认状态：待用户确认

- 对象编号：`OB-C026-002`
- 场景：长期浅聊后，她第一次主动给出接下班机会。
- 女生原话：“那晚上你可以接我下班了。”
- 讲师判断/处理：窗口突然变大，应从闲聊切执行，不再浪费。
- 发现的缺口：缺少不同强度的接法。
- 候选补足：A 直接确认：①“几点下班？”②“位置发我。”③“我过去要二十三分钟。” B 带计划：①“我接你，顺路吃点东西。”②“下班先上车，再决定去哪。”③“我提前十分钟到。” C 坏坏认领：①“这次接你，下次你欠我一顿夜宵。”②“终于给我一个像样的任务。”③“可以，今晚暂时归我接管。” D 给退路：①“如果临时加班提前说。”②“太累就直接送你回去。”③“不方便见也别硬撑。”
- 后续分支：给位置→复述 ETA；问去哪里→给两方案；犹豫→降为送回；临时取消→不争，另定具体时间。
- 补足理由：把罕见行动窗口立即落地。
- 来源边界：接下班与距离确认来自课程；说法为候选。

## SUP-C026-003：临出发她说累、不想收拾

- 对象编号：`OB-C026-003`
- 场景：已约好，她突然觉得洗漱、护肤、带睡衣太麻烦。
- 女生原话：类似“要不等我休息的时候吧，我不想收拾，好累。”
- 讲师判断/处理：临门下头不一定是真拒绝；给台阶、减少准备成本，再增强执行感。
- 发现的缺口：需区分客观麻烦、情绪犹豫、真实取消。
- 候选补足：A 降低成本：①“洗漱用品这边都有。”②“不用特意收拾，带手机就行。”③“困了就直接休息。” B 理解再推进：①“知道你累，我已经准备出门了。”②“来回折腾是烦，我过去接你。”③“今晚不安排复杂内容。” C 坏坏带领：①“别在最后五分钟变卦。”②“收拾任务取消，人到就行。”③“我车已经醒了，你别又睡回去。” D 接受取消（明确不想来）：①“那就改休息日。”②“别勉强，给我下一次具体时间。”③“今天取消可以，不留模糊账。”
- 后续分支：重新答应→给 ETA；继续列麻烦→逐项降成本；沉默→暂停出发；明确取消→定新日或结束。
- 补足理由：把最后一刻阻力拆成可判断分支。
- 来源边界：台阶与强推进来自课程；候选句为扩展。

## SUP-C026-005：被指责欺骗和白嫖时先认怂

- 对象编号：`OB-C026-005`

- 女生原话：类似“你们就是欺骗和白嫖。”
- 讲师判断/处理：不要对抗；先认怂、道歉，再做必要事实解释。
- 发现的缺口：缺少道歉、澄清、停止和后续修复的梯度。
- 候选补足：A 先道歉：①“这件事让你感觉被欺骗，是我的问题。”②“对不起，我没有处理好信息边界。”③“你生气有理由，我先听完。” B 澄清事实：①“照片确实用于工作展示，我应该提前说明。”②“哪些内容让你最不舒服，你指出来。”③“我只解释事实，不跟你争感受。” C 停止刺激：①“你现在不想继续聊，我先停。”②“相关内容我不再转发。”③“需要我删除或更正的部分列给我。” D 修复关系：①“等情绪下来我再正式说一次。”②“如果你愿意，我们把边界重新讲清。”③“你不接受修复，我也尊重。”
- 后续分支：继续骂→不反击；给事实点→逐项回应；要求停止→停止；愿意沟通→约定边界。
- 补足理由：复现课程明确的认怂处理链。
- 来源边界：认怂道歉原则来自课程；具体句为候选。

- 讲师判断/处理：先相处，把脸暂时往后放；恋爱和结婚分线，观察性格、情商与生活协作。

- 后续分支：发现相处舒服→继续观察；持续排斥→停止推进；家庭催促→给明确观察期限；信息不足→安排新的真实相处场景。
