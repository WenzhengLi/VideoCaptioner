# C026 案例动作链

## CASE-C026-001：低信息聊天到见面

长期短回复但不断线 → 不因内容浅判死刑 → 她给位置并说可接下班 → 直接确认路程、时间和接送 → 她临时说累、不想收拾 → 给“这边都有”的台阶并说明已经出发 → 她重新配合 → 见面过夜 → 次日接送和后续报备维持关系。

## CASE-C026-002：指责与告密

女生以“欺骗、侵犯、白嫖”指责 → 不争逻辑高下 → 先认怂、道歉 → 用工作场景照片解释展示内容 → 停止刺激对抗；课程再转入群内分享边界和熟人圈传播提醒。

## CASE-C026-003：颜控与订婚犹豫

学员第一次整天相处后仍对脸犹豫 → 讲师不催订婚也不只看脸 → 建议先相处 → 把恋爱吸引与婚姻协作分线 → 观察情商、性格、生活能力和家庭关系后再决定。
