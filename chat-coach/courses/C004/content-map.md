# C004 开放式内容地图

状态：整课第一遍扫描完成，等待用户确认。

## 一、课程整体在讲什么

C004 把“外卖到家”拆成一条连续决策链：先通过工作时间、休息日和日常活动判断当日窗口；用生活物品形成自然共处画面；女生一旦进入行动准备，就会连续确认照片真实性、体型、过夜、清洁、睡衣、牙刷和防护。课程强调这些不是无意义的刁难，而是她去陌生人家前必须完成的现实检查。第二案例又展示：即使聊天和前置顾虑都解决了，男生在临门一脚不回复，窗口仍会迅速掉下去；此时电话、亲自接和诚意动作比继续打字有效。

```text
信息交换确认属性与时间
→ 从她主动给出的空档识别窗口
→ 用具体生活物品发起模糊邀约
→ 将“这就来”转成位置、时间和车辆的行动验证
→ 耐心解决真实性、舒适度、清洁和准备问题
→ 她犹豫时分辨速度顾虑还是执行麻烦
→ 临门失误后用电话和亲自接修复
→ 见面后继续维护，必要时可转长期
```

## 二、章节与内容变化

### 05:10–12:30：工作属性、时间与当天窗口

- 女生周末早起上班，学员先问工作和作息；
- 她不正面回答具体工作，课程要求不要直男式追问；

- 女生说每天上班，但明天是下午班；课程读成“今天有空，你可以安排”；
- 立刻问今天几点下班，而不是继续聊职业。

### 12:30–16:30：果冻画面、模糊邀约和行动验证

- 女生主动追问男生在做什么，窗口仍在；

- 女生说“这就来”，学员没有直接当真，而是问位置、距离和现在是否方便；
- 当叫车动作出现，她又说“别太急”，暴露此前仍含嘴炮和犹豫。

### 16:30–24:00：累、照片本人和体型测试

- 女生说本来想睡觉、太累，学员提供路上和到家都能休息的画面；

### 23:20–32:30：过夜和生活用品题库

- 女生直接问是否过夜，课程解释为两层：确认关系安排，也决定要不要带卸妆、隐形眼镜、化妆品等；

### 26:00–38:30：“你真专业 / 老手”与准备成本

- 家里用品过于齐全，会触发她对其他女人和卫生的担心；

- 她说洗头洗澡化妆要两小时，说明重视见面，也形成实际等待时间。

### 42:00–46:21：不要整得像谈恋爱与执行落地

- 喜欢应当在观察到她的具体价值后产生，而不是先想谈恋爱再随便找个人。

## 三、课程核心体系

### 时间信息不是闲聊

“明天下午上班”“最近没上班”“染完头发没安排”都是可执行窗口。课程要求把时间信息转成下一步，而不是停在作息讨论。

### 模糊邀约要进入行动验证

“一起吃果冻”“一起撸猫看电影”负责打开画面；她说“这就来”后，必须用位置、距离、现在方便、电话和车辆验证，防止把嘴炮当行动。

### 生活用品问题是到场条件

卸妆、睡衣、牙刷、毛巾、防护和是否过夜，分别解决舒适、卫生、第二天安排和安全问题。课程要求耐心回答，并把准备成本降到“人来就行”。

### 短期框架也需要分寸

双方明牌不代表可以把所有表达都变露骨；“怎么快乐”要让双方听懂，又保留体面和情绪。女生说别像谈恋爱，说明浪漫强度要降而非继续加码。

### 犹豫时替她做决定

当时间已空、地点可行、前文有窗口，只是“晚上再定”，课程倾向继续踩油门。但替她决定仍以她持续回应和没有明确拒绝为条件。

### 临门执行权重大于前面话术

女生已经准备和到达时，二十分钟不回能毁掉前面全部铺垫。修复要用电话、真实理由、亲自接和立即行动，不能继续靠漂亮文字。

### 邀约要回答她为什么值得来

共同兴趣、体验、现实帮助和她关心的事情，是对方视角的筹码；单纯“我喜欢、我觉得你好看”只表达男生需求。
