# C004 案例动作链

## 案例一：当天添加、当天到家

### 链路 A：作息信息到当天窗口

```text
女生周末早起工作
→ 男生询问属性，但她不正面回答
→ 不追问具体职业，改为分享自己作息
→ 女生说明天是下午班
→ 课程读成今天有空
→ 直接问今天几点下班
```

### 链路 B：果冻模糊邀约

```text
男生晒快递和果冻
→ 女生主动注意并提出来
→ 男生说一起吃
→ 说明自己独居、父母在老家
→ 女生说这就来
→ 男生开始问位置、距离和是否马上方便
```

### 链路 C：嘴炮与真实行动

```text
女生口头说马上来
→ 男生准备叫车
→ 女生说别太急、自己很累
→ 课程判断前一句仍含嘴炮和犹豫
→ 男生提供路上和到家都能休息
→ 继续验证是否真实准备
```

### 链路 D：真实性与体型预期

```text
女生问照片是不是本人
→ 防止被假照骗到陌生地点
→ 女生说自己很胖、真人有差异
→ 需要确认男生是否接受
→ 男生顺着表示真人合理差异没关系
→ 不在见面前制造外貌审判
```

### 链路 E：过夜准备题库

```text
女生问是否过夜
→ 决定卸妆、隐形眼镜、化妆和第二天安排
→ 男生说可以留宿但不强求
→ 女生继续问睡衣、牙刷、毛巾、漱口水和防护
→ 男生逐项解决
→ 准备成本降到人来就行
```

### 链路 F：专业 / 老手与卫生疑虑

```text
家中用品齐全
→ 女生说你真专业、像老手
→ 背后担心其他女人、共用物品和卫生
→ 用自用物品、新毛巾、临时购买解释
→ 防护可外卖或由她自带
→ 让她保留掌控感
```

### 链路 G：短期框架与落地

```text
男生提前描绘搂腰等
→ 女生：别整得像谈恋爱
→ 男生说自己有分寸
→ 她洗头洗澡化妆、给电话和车牌
→ 下雨时安排下楼接
→ 到家相处
```

## 案例二：凌晨失眠到次日晚间见面

### 链路 A：孤独共鸣到含蓄明牌

```text
女生凌晨睡不着、分享视频
→ 男生从哭、肩膀和抱抱建立情绪
→ 问位置、距离和休息时间
→ 女生说一个人不知道做什么、对什么都没兴趣
→ 男生说两个人在一起就不会无聊
→ 一起做有感觉的事
```

### 链路 B：文字与电话选择

```text
双方已明确孤独和见面意向
→ 讲师建议直接电话
→ 通过声音确认状态、解释顾虑、敲时间地点
→ 学员因家中另有女生只能继续文字
→ 约定次日晚间
```

### 链路 C：晚上再定与继续踩油门

```text
女生染头发、说晚上再确定
→ 实际时间仍空，前文窗口存在
→ 课程判断是犹豫而非明确拒绝
→ 男生不把决定全部交回
→ 继续询问进度、说不远、先见再看
```

### 链路 D：第一次去家与态度测试

```text
女生问是不是去你家、第一次见不太好
→ 男生不做长篇自证
→ 说楼下先见，回家就是第二次
→ 一回生二回熟
→ 用波澜不惊的态度通过框架确认
```

### 链路 E：怎么快乐与速度顾虑

```text
女生问怎么快乐
→ 不能直接露骨，也不能答非所问
→ 用“萧亚轩的快乐”等双方都懂的含蓄表达
→ 女生说不想发展太快、有点虚、太唐突
→ 男生用感觉、双向奔赴和只邀请重要的人推进
→ 接受先楼下走走的台阶
```

### 链路 F：临门失联与修复

```text
女生已准备、到附近等待
→ 男生因另一女生在家二十分钟不回
→ 女生从兴奋转为改天、以后有机会
→ 课程给失联操作零分
→ 文字解释不足
→ 应打电话听情绪并现场拉回
→ 必要时亲自去接，补诚意
```

### 链路 G：到场拆台与维护

```text
女生到小区
→ 保安说男生经常带女生回来
→ 女生质疑特殊性
→ 男生说只有重要的人会来
→ 她反问重要的人很多
→ 继续用后续行动和日常维护修复
→ 课程判断可转长期
```

## 课后原则：从对方价值出发邀约

```text
只说你漂亮 / 我喜欢 / 我想谈恋爱
→ 都是男生自己的需要
→ 女生没有见面的理由
→ 改为共同宠物、电影、滑雪、工作、现实帮助或共同经历
→ 让邀约回答“她为什么值得来”
```
