# C024 案例动作链

## CASE-C024-001：对抗路拉扯至见面

1. 普通开场后过快推“一起待” → 女生质疑目的和女性资源 → 男生进入解释与框架对抗。
2. 女生表达自己普通、距离感强 → 课程建议降低价值压迫，增加真实日常和可接近感。
3. 女生追问是否已婚、是否花 → 识别她在核验风险 → 回答后回到具体相处，而不是无限自证。
4. 男生用“有机会、回来、周末”模糊推进 → 女生要求提前计划 → 改为具体日期、时间和内容。
5. 围绕第一次见面、去家里与谁付出持续拉扯 → 讲师判断她更吃被照顾表达 → 调整为带计划的柔和推进。
6. 见面当日堵车、爆胎、延误 → 及时同步位置与变化 → 重新协调 → 最终见面。
