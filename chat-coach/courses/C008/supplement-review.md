# C008 候选补足稿

## SUP-C008-001：没洗头、有点丑但仍问时间
- 对象：`OB-C008-001`；课程判断：形象顾虑不是拒绝，主动问时间和车程才是行动信号。
- A 降形象成本：①“没事，今天不是选美。”②“你舒服来就行。”③“素颜版本也可以认识。”
- B 坏坏接反差：①“先把丑话说前面，看来真人有惊喜。”②“没洗头也敢来，窗口挺诚实。”③“我先验收你说的有点丑。”
- C 给短见方案：①“先来坐半小时，不舒服就走。”②“到附近喝一杯，别做大工程。”③“先见面，下次再精心打扮。”
- D 行动落地：①“你几点出发？”②“车程多久，到了我接你。”③“位置发我，我把入口说清楚。”
- 分支：继续问细节→“那就十二点半，我把入口发你。”；只重复丑→“先见半小时，别为造型折腾。”；明确不来→“好，今晚不催，改天你方便再说。”来源：原顾虑与行动来自课程，替代表达为候选。

## SUP-C008-002：人多我社恐
- 对象：`OB-C008-002`；课程处理：人数说两三个，不能夸大场面。
- A 如实降人数：①“就两三个人。”②“不会让你进陌生人大局。”③“你来主要见我。”
- B 提供退出：①“待不舒服我陪你出去坐。”②“先露个面，随时能走。”③“人多我们就换安静处。”
- C 坏坏照顾：①“社恐席位给你留我旁边。”②“你不用营业，我负责接场。”③“只需要认识我，其他人算背景。”
- D 改私下见：①“真不想进场就单独喝东西。”②“我出来接你，先在外面聊。”③“今晚不合适，另定安静见面。”
- 分支：问几个人→“加我一共三个人。”；到场紧张→“我们先出去坐会儿。”；明确不喜欢酒吧→“那改成单独喝东西。”来源：人数与社恐来自课程。

## SUP-C008-003：第一次只验货，不必一次做完
- 对象：`OB-C008-003`；课程判断：多人酒吧难兼顾，第一次能建立好印象，后续仍可继续。
- A 现场专注：①“先认识你，今晚不赶流程。”②“我们出去坐会儿。”③“你觉得舒服最重要。”
- B 轻暧昧：①“真人比照片更有感觉。”②“你坐我旁边，我才有参与感。”③“今晚先把陌生感拿掉。”
- C 留后续：①“这里太吵，下次单独见。”②“今天验收通过，下一次我认真安排。”③“没聊完的留给安静场。”
- D 到家维护：①“到了告诉我。”②“今天见你比场子本身有意思。”③“早点休息，改天继续。”
- 分支：她主动留→“那陪我再坐一会儿。”；离开后热情→“下次我们单独见。”；冷淡→“到了早点休息，等你有状态再聊。”来源：验货和后续维护来自课程。

## SUP-C008-004：“慢慢来”不是单一含义
- 对象：`OB-C008-004`；课程判断：可能是长期倾向、矜持、筛选或惯性说辞，不能一棒定死。
- A 接受节奏：①“可以慢，但别停。”②“我不催结果，先看相处。”③“慢热没问题，保持真实投入。”
- B 点破拖延：①“慢慢来不能等于一直养着。”②“我接受过程，不接受无限模糊。”③“你给行动，我给耐心。”
- C 表达加热：①“我也慢热，只对喜欢的人加热快一点。”②“总得有一个人先主动。”③“感觉到了，我不会故意装冷。”
- D 给验证：①“先再见一次，看是不是更舒服。”②“用见面验证，不用口头定义。”③“下一次相处完再谈节奏。”
- 分支：她解释慢热→“我接受，那我们先再见一次。”；只要投入不行动→“等你能给具体时间再继续。”；主动见→“你给行动，我就认真安排。”来源：慢慢来的多重判断来自课程。

## SUP-C008-005：“之前也有人这么说”
- 对象：`OB-C008-005`；课程处理：不替别人负责，强调自己始终如一和实际行动。
- A 区分个体：①“别人怎么做我改不了。”②“相同一句话，不代表同一个人。”③“你可以观察我，不必先判我。”
- B 行动证明：①“看我是否按约出现。”②“我少保证，多兑现。”③“你把标准说清，我让行动回答。”
- C 坏坏反问：①“你把前任面试题发给我了？”②“先别拿旧剧本给我定角色。”③“我可以接受观察，不能替别人背锅。”
- D 承认她谨慎：①“被画过饼，谨慎正常。”②“你不用马上信。”③“先给我一次验证机会。”
- 分支：她讲旧经历→“你最介意他后来哪种变化？”；继续泛化→“先看我这次会不会兑现。”；明确不信→“不用现在信，先慢一点观察。”来源：原对话来自课程。

## SUP-C008-006：家里管得严与门禁
- 对象：`OB-C008-006`；课程判断：摄像、定位、家人查岗是客观约束，也塑造她的谨慎。
- A 尊重时间：①“两点前回就按两点前设计。”②“我提前送你。”③“别让你回去再解释半天。”
- B 短见：①“九分钟距离，先见一小时。”②“下班附近坐会儿。”③“今天短一点，下次留完整时间。”
- C 坏坏理解：①“原来你出门自带审计。”②“自由时间珍贵，我不浪费。”③“查岗严格，见我还挺有诚意。”
- D 长期铺垫：①“下次白天安排，不总卡门禁。”②“提前报行程，比临时跑顺。”③“先让家人知道你和朋友正常见面。”
- 分支：她给极限时间→“两点前回，那十一点半见。”；家人临时催→“我现在送你回去。”；无法出门→“好，今天不折腾，改白天。”来源：家庭约束来自课程。

## SUP-C008-007：她问“这么快吗”
- 对象：`OB-C008-007`；课程处理：把快慢改写成喜欢后的正常美好与浪漫画面。
- A 浪漫画面：①“起床有人做早餐，为什么一定要等很久？”②“喜欢的人在旁边，普通早晨也会甜。”③“亲亲抱抱和早餐，是相处不是考试。”
- B 降低预设：①“我在描绘可能，不是逼你交结果。”②“先见面，快慢现场感受。”③“你觉得快，我们就把动作拆小。”
- C 坏坏推进：①“感觉来得快，不代表人很随便。”②“喜欢还要故意装慢，那多累。”③“你负责踩刹车，我负责告诉你前面风景。”
- D 行动验证：①“先一起吃顿早餐前的夜宵。”②“先完成一次舒服相处。”③“别争定义，用下一次见面验证。”
- 分支：她接浪漫→“那把第一顿早餐留给我们。”；仍怕快→“先见面，不谈过夜。”；明确不愿→“收到，这个方向停在这里。”来源：早餐与快慢辩论来自课程。

## SUP-C008-008：粘人代表重要性
- 对象：`OB-C008-008`；课程说法：有事说事、无事退朝不像关系；区分“其他男人”和自己。
- A 支持亲近：①“我喜欢有分享欲的人。”②“想我就来，不用端着。”③“关系里有点黏才有参与感。”
- B 保留空间：①“黏可以，各自忙时也要自在。”②“亲近不是全天考勤。”③“有需求直接说，不靠猜。”
- C 坏坏占有：①“你黏人的名额我先预订。”②“对别人高冷，对我可以破例。”③“想退朝可以，先给本王报备。”
- D 反筛选：①“如果只享受被黏却不回应，我不玩。”②“亲近要双向。”③“我会热情，也看你的投入。”
- 分支：她说自己黏→“那我期待你熟了后的样子。”；怕烦人→“想分享就来，我不嫌。”；只索取→“我会回应，也需要你回应我。”来源：粘人讨论来自课程。
