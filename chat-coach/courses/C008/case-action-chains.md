# C008 案例动作链

## CASE-C008-001

临时邀约 → 她报没洗头/丑/社恐 → 降低人数与形象成本 → 她打车到酒吧 → 现场先认识与验货 → 回去维护 → “慢慢来”长期框架 → 点破拖延但接受其顾虑 → 用偏爱、行动和爱情画面升温 → 家庭管控/太晚/安全逐项处理 → 给私下活动筹码 → 到达 → 现场节奏不足时退回喝酒聊天 → 再推进 → 次日维护。

## CASE-C008-002

首次接下班被拒 → 不强求 → 猫、粘人和关系观信息交换 → 区别于其他男人 → 星座偏见改写成对外高冷、对自己人偏爱 → 再邀约 → 处理晚、工装、距离与回程 → 她到达 → 见面与后续联系。
