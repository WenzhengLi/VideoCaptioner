# C021 案例动作链

## CASE-C021-001：汕头女生，不温不火到咖啡邀约

持续但偏短回复 → 沿城市、旅行和热门剧增加共同画面 → 多句并发软试探“一起看” → 她避开邀约，停止逼问并换题 → 补名字与了解欠账 → 她夸帅、关心作息，顺势拉近 → 她连续主动，讲师判断投入反转 → 忙碌时拿位置等行动信息 → 分享食物、演唱会并描绘一起待着 → 小邀约给活动选择 → 她选咖啡 → 确认区域、口味与到达方式 → 见面落地。

关键门禁：持续回复不等于高窗口；软邀约未被接住时不强迫正面表态；夸赞、关心和连续主动出现后才升级。男生多次长时间不回被讲师明确判为问题。

## CASE-C021-002：成都女生，高窗口转安全感

躲被窝开场 → 女生主动“同步”形成大窗口 → 日常关心、猜名字、星座形成默契 → 讲师判断吸引已解决 → 不急于猛约，转而了解空窗、前任和关系倾向 → 识别她对长期关系与归属感的需要 → 前任原因简洁交代，不沉浸细节 → “珍惜当下”引导“缘分持续” → 询问休息时间 → 她用本地小吃和接待给出未来见面接口。

关键门禁：高窗口不自动等于可立即落地；先判断对象、需求和安全感缺口。前任话题只为解释未来风险，不把聊天变成前任复盘会。

## CASE-C021-003：报备保温到见面执行

女生索要晚安 → 男生顺势索要早安 → 应酬喝酒转未来小酌 → 微醺与浪漫构成未来场景 → 双方持续报备维持温度 → 明确接机日期、地点和安排 → 长时间消失造成温差，再恢复维护 → 接机计划变化后重新协调 → 女生主动来找 → 用休息、看电视落到共同空间 → 打车、电话、到达执行 → 见面后继续互动维护。

关键门禁：先有吸引信号再描绘升级场景；现实说法停在课程允许直接表达的范围。邀约需有时间、地点、活动，并能在计划变化时重新落地。
