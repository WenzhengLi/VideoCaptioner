# C025 案例动作链

## CASE-C025-001/002：展示面到酒店外卖

生活化高尔夫/机车内容 → 女生对生活方式形成预期 → 酒店外卖上门；发生关系后不只停在结果，继续判断性格与长期可能，并转广州主场邀约。

## CASE-C025-003：见面方式比较

门徒汇报“见十拿下三” → 比较外卖上门、外面约会、接送带回 → 讲师判断不同窗口需要不同现场能力 → 用大量真实约会训练而非只背话术。

## CASE-C025-004：新疆女孩

得知只停留一周 → 快速描绘“甜甜待一起” → 她质疑是否对所有女生都这样 → 用“不滥情”简短刹车并换到老家/家人 → 谈感情观和“外表顺眼、氛围三观” → 表达对她有感觉与特殊性 → 学她的“阿达西”进入共同语言 → 已约后反复强化共同画面 → 夜店发出行动窗口 → 直接拿位置和时间 → 明确只接她一人 → 到场接人。
