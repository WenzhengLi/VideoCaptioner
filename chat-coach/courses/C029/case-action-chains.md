# C029 案例动作链

## CASE-C029-001

一对一邀约已定 → 她临时说带狗/闺蜜 → 判断尴尬缓冲还是测试边界 → 明确原安排或改天 → 不用害怕失去邀约而层层妥协。

## CASE-C029-002

加好友后她主动催聊 → 交换位置、一脚油门拉近 → “不缺女朋友吧”高价值测试 → 保持原有人设并表达关系选择标准 → 她讲前任、生日受伤与付出 → 先共情，再植入纪念日、照顾与专一画面 → 她主动问类型、担心不理她 → 接住窗口 → 确认吃饭后过来、是否接她等执行。

## CASE-C029-003

线上口头默认男女朋友 → 高铁站首次见面且同事在场 → 学员直接出现造成集体尴尬 → 同事离开后急牵手 → 她生气 → 学员让她走 → 无后续。课程替代链：远处询问并等单独会合 → 接行李建立带领 → 观察线下反馈 → 再逐步升级；出现生气时先处理情绪，不立即逃离。
