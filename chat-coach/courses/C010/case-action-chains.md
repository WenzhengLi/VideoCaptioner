# C010 案例动作链

## CASE-C010-001
- 边界：`SEG-C010-000193`–`SEG-C010-003176`
- 案例概况：佛山八七年医美单亲妈妈老资源，从朋友圈评论切私聊，经长期价值观、异地顾虑、语音和订票推进奔现

### OB-C010-001｜评论成都疫情后的切私聊
1. 场景/原话：成都那边有疫情吗？
2. 讲师判断：她主动评论成都相关朋友圈，说明成都或发圈人进入了她的注意范围；讲师把评论视为老资源重新打开的入口。
3. 课程动作：不在评论区长聊，切私聊说明成都情况，并顺势问她是否想来玩。
4. 下一节点：观察对方是否增加信息、主动确认或行动配合；若没有新增投入，不把单句继续放大。

### OB-C010-002｜没有闲情雅致后的继续推进
1. 场景/原话：我没有那个闲情雅致。
2. 讲师判断：表面否定旅游，但仍持续回复；讲师没有把它当红灯，而是继续补充自己在成都定居、也会去佛山谈项目的现实交集。
3. 课程动作：用工作和城市交集降低异地感，再继续了解她的时间和可行动性。
4. 下一节点：观察对方是否增加信息、主动确认或行动配合；若没有新增投入，不把单句继续放大。

### OB-C010-003｜等你回来有机会了解一下
1. 场景/原话：等你回来可以有机会了解一下。
2. 讲师判断：讲师认为“有机会”遥遥无期，真正的机会需要男人主动把握并落到时间。
3. 课程动作：从抽象机会推进到具体时间，并提出订票来成都两天的新体验。

### OB-C010-004｜为什么不是你飞回来见我
1. 场景/原话：你飞回来见我不行吗？

3. 课程动作：给出公司离不开的现实理由，同时承诺她来后可以陪伴、照顾和共同体验。
