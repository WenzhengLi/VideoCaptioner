# C030 案例动作链

加好友后短招呼 → 忙碌漏回时礼貌解释 → 问在干嘛 → 她分享夹娃娃 → 围绕少女心、数量和地点连续接题 → 同城距离 → 打雷/害怕 → 描绘另一半陪伴 → 她测试“不缺女生吧” → 回应单身、工作、圈子与归属感 → 她继续问工作、陪伴、前任和关系标准 → 价值观共鸣加深 → 她展示纸条与年轻追求者 → 不进入低位竞争 → 表达有感觉会主动但不是陌生场合乱搭讪 → 提出见面/一起待 → 她给安全、随便、强迫和了解顾虑 → 给态度与持续了解框架 → 次日问接人时间 → 确认出门、ETA、堵车 → 见面。

课程限制：高分女生的开聊需要更连续；明确拒绝不应用“心态/话术”硬顶，实力与展示面是前提。
