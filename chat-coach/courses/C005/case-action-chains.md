# C005 案例动作链

## 案例一：连续黄灯到窗口反转

### 链路 A：黑漆漆与孤独画面

```text
女生主动招呼并说黑漆漆
→ 课程判断包含孤独状态
→ 男生描绘有人在身边会浪漫
→ 女生认同
→ 询问位置和距离
→ 提出一起待和次日看书
```

### 链路 B：第一次去家连续黄灯

```text
女生：才认识去家里不好、明天有事
→ 男生不正面回答好不好
→ 用在家舒服、喜欢聊天状态、明天送她侧面处理
→ 女生继续说麻烦、准备睡、太快、害怕
→ 男生逐轮缩小结果、强调看感觉和不强求
```

### 链路 C：敏感与画饼

```text
女生说被动、敏感、想得多
→ 男生说正因她被动自己才主动
→ 女生说先别谈照顾，没实现就是画饼
→ 男生承认无行动确实是画饼
→ 用叫车、下楼接和次日送她证明
```

### 链路 D：喜欢太快与颜控

```text
女生质疑喜欢飘忽、只看照片
→ 男生定义喜欢为瞬间想行动
→ 区分理性选择和真实感觉
→ “我对自己很了解”建立稳定自我认知
→ 把照片扩展为聊天、整体感觉和价值观
```

### 链路 E：害怕、过夜与特殊性

```text
女生害怕躺在一起
→ 男生说不强求，只是抱着感受彼此
→ 很多事不用一晚做完
→ 女生问是否经常带女生回家
→ 男生反问家又不是公共场合，谁都能来吗
→ 建立她是例外的感觉
```

### 链路 F：底层安全感补作业

```text
多轮问题都被回答
→ 女生仍反复犹豫
→ 讲师判断底层可能缺安全感
→ 不再只换话术
→ 回到工作、生活、真实状态和稳定互动补前文
```

### 链路 G：主动追问后的窗口反转

```text
男生因生病或其他安排后撤
→ 女生主动追发、问为何不找她
→ 关心喝水、饮食和发烧
→ 窗口从防御转为主动
→ 开始问照片反差、喜欢程度和过夜准备
→ 发位置、电话并线下落地
```

## 案例二：酒吧认识到酒店现场

### 链路 A：大量报备和关心

```text
女生离开酒吧
→ 解释已到家、第一次去酒吧、烟熏耳鸣和运动疲劳
→ 提醒男生少喝、早点回
→ 课程判断信息量、解释和关心代表明显窗口
→ 即使后面只回嗯，也不能孤立定性
```

### 链路 B：去干嘛与明暗牌

```text
男生邀她过来
→ 女生问去干嘛
→ 男生给电影、聊天、吃东西、吹空调和有感觉的事
→ 女生确认是不是去家并点破小九九
→ 男生不否认也不长篇自证
→ 女生说没事，双方理解关系目标
```

### 链路 C：持续报备与相亲危机

```text
女生报备飞盘、酒吧、位置和结束时间
→ 半夜主动找男生
→ 次日说去相亲
→ 课程判断在制造危机感和展示其他选择
→ 男生继续要求具体时间和位置
```

### 链路 D：另一个男人与选择权

```text
同体系另一个男人也邀她
→ 女生告诉男生并观察反应
→ 男生说选择权在她
→ 不立即进入争抢
→ 可后撤让她自己做选择
```

### 链路 E：酒店下楼但仍等待

```text
女生到达后发现是酒店
→ 男生又因房内其他女生接应混乱
→ 她下楼、情绪下降
→ 但没有离开，仍在附近等
→ 课程把等待视为强行动窗口
→ 先在大厅恢复情绪
```

### 链路 F：大厅距离测试与年龄问题

```text
女生坐对面
→ 男生让她坐近
→ 她愿意移动，第一轮现场测试通过
→ 女生问年龄并说不谈弟弟
→ 男生用经历决定成熟度重新定义
→ 她说相亲对象看不上
→ 男生框架为过来说明看得上自己
```

### 链路 G：上楼后的长期人设龟缩

```text
女生口头说不上楼
→ 却让男生拿包、跟随电梯和走廊
→ 进入房间后不换鞋、坐远、说只坐会儿
→ 课程判断她知道短期含义，但长期人设开始龟缩
→ 用环境理由、坐近、稳定拉近和观察行动处理
→ 讲师明确不建议模仿极端发脾气
```
