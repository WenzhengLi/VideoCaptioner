# C022 案例动作链

## CASE-C022-001：汕头白富美长期跟进

1. 初期回复一般 → 更新更完整的生活展示 → 女生回复质量上升。
2. 交换旅行、烟花与生活内容 → 确认双方是否有共同体验 → 描绘以后一起去、一起看的画面。
3. 女生问是否“对很多女生都这样” → 识别为真实性与认真程度测试 → 不急于自证，区分有经历与滥情。
4. 她提出异地、停留时间和关系去向顾虑 → 先承认现实成本 → 提供继续了解的筹码，不强压结论。
5. 第一次推进未完全落地 → 收回邀约压力，继续日常保温 → 待主动和信息量回升后再推进。
6. 确认愿意见面 → 从模糊意向进入时间、地点、活动、接送等执行细节 → 见面前持续保温。

讲师的动作不是固定句式，而是“读窗口—推进—回收—补筹码—再推进—执行确认”。
