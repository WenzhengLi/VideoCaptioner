# C031 案例动作链

短表情/短回复 → 不误判关闭 → 寒暄后直接问位置和结束时间 → 提出一起待 → 她追责上次没回 → 用真实修机照片解释 → “聊天记录重要”表达特殊性 → `miss you` 升级 → 她问其他女生 → 给偏爱回应 → 她坚持白天见、今晚太快 → 描绘今晚与以后白天都会陪 → 她要求接、质疑理由多和不在意 → 多轮真诚解释 → 重复测试时短暂立态度 → 她继续对抗 → 软化说明工作先处理好是为了完整陪伴 → 发位置、去接、邀约落地。

课程强调的推拉节奏：先软 → 重复测试时硬一次 → 不连续硬顶 → 再软并给行动筹码。课尾明确说，若跳过“态度”直接一味付出，容易落入她不尊重的低位。
