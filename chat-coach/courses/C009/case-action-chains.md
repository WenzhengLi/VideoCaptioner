# C009 案例动作链

## CASE-C009-001

浅聊旧线索 → 抵达她城市后集中进攻 → 她说全天有空/可调休 → 男生表达为她奔赴 → 她要求第一次见回家 → 先说可以 → 后续用孤独、喜欢和相处画面重建 → 她说不喜欢快、不想随便、要多见几次 → 课程用长期案例、感觉优先和行动证明逐层回应 → 她质疑为何对她有感觉、是否多线、主动是否过猛 → 补资料观察、具体欣赏、生活分享和专注时间 → 她要求更多生活关心 → 加入追剧、报备、梦境和日常 → 她让男生回来并提出接送 → 男生订票返回 → 见面后继续约下班 → 处理明天工作、早睡、用品和地点 → 她给定位并前来 → 完成落地。

## 同场新对象分支

女生主动加好友 → 互夸外形并问故事 → 讲师用孤独与美好故事拉近 → 问下班时间 → 她持续投入 → 给私下活动与接送 → 临出门犹豫 → 继续解决休息和用品 → 她给定位并前来。
