# C027 案例动作链

## CASE-C027-001

共同旅行/地点 → 建立相似生活感 → 提出一起待 → 女生连续讲“不熟、尴尬、应先外面见” → 课程读取长回复与持续解释为窗口仍在 → 继续推进后发现理性阻力未消 → 减速或切价值观、名字、生活和感情观 → 女生主动给回程时间 → 确认东站、位置、穿着、到达时间 → 见面。

关键点：不是把“长回复”等同于同意，而是判断她仍愿意投入；连续强冲无效时需要改变说服材料。

## CASE-C027-002

表情包互怼 → 年龄嫌弃 → 不急于自证，反收集朋友圈/形象反馈 → 她偷表情包 → “为什么不偷偷喜欢我”推近 → 不立即低位承接 → 用她的表情包反打 → 未来孩子/基因画面 → 她质疑是否对多人说过 → 不长篇解释，保持推拉。
