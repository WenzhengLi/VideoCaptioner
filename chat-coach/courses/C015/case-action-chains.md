# C015 案例动作链

## CASE-C015-001：九眼桥求救式窗口

```text
女生发来旁边男人请求“下次找你”的语音
→ 结合凌晨、喝酒和拖延语气判断她想脱身
→ 问发生什么/是否有人缠她（课程复盘建议）
→ 问位置、提出去接
→ 她口头说不用，但继续回答喝完与去向
→ 男生再次确认接走
→ 她三次问接走后去哪
→ 用陪伴、热水、撸猫、聊天、亲密画面解释气氛
→ 她接受安排
→ 男生到场接走
```

## CASE-C015-002：高窗口破解顾虑并当晚见面

```text
她主动开场/表情/共鸣/连续提问/夸帅
→ 讲师判巨大窗口
→ 男生建立事业忙、有标准的人设
→ 问休息时间并提出一起待
→ 她说不合适、不能随便过夜
→ 尊重她人设，也表达自己不随便
→ 她说慢慢相处、不急于肢体
→ 心灵感受先于身体 + 即使不做什么也开心
→ 她担心一夜情/炮友/第一次发生关系
→ 重复长期可能、责任、相互和真实欲望
→ 不接受无限拖延，强调当下感觉
→ 她说明晚去机场，只能回来再见
→ 男生提出今晚、早餐、不要错过
→ 她发位置、电话、准备、下楼
→ 叫车到达滨河湾并过夜
→ 到场后仍需维持温柔、照顾和关系体验
```
