# C016 案例动作链

## CASE-C016-001：删好友后重开到异地奔现

1. 女生识别轰趴套路并删除 → 学员保留联系方式，隔一年按新认识重新加 → 不纠缠旧冲突。
2. 女生持续闭塞回复 → 学员继续少量提供话题 → 讲师判断再持续会变陪聊。
3. 学员直接表达“有你才算丰富”、邀请来成都 → 女生从单字/表情变成追问真假 → 窗口开始松动。
4. 女生说带女朋友、自己不轻易谈 → 学员表达喜欢、有标准、不将就 → 女生开始输出长期需求与价值观。
5. 女生主动报备、关心航班、讲成都故事 → 学员顺着她的感受共鸣 → 再提出什么时候来。
6. 女生对直邀“不理解” → 学员不硬顶，退回询问休息时间 → 保留未来执行口。
7. 女生发烧缺药 → 学员主动找地址、调南宁资源送药 → 抽象“照顾”变成实际证据。
8. 女生反复说各有生活、暂未定时间 → 学员持续锁“休息时来”，同时保持低频日常 → 她主动报告可能提前放假。
9. 放假后女生说太突然 → 学员先表达失望，再接受机票未看等事实 → 提供订票和具体安排。
10. 女生担心怎么安排、外貌、被卖、寒冷 → 学员逐项给两三天安排、生活用品、火锅/在家选项、接机和安全感 → 身份证出票。
11. 落地后按三楼十一号门接机 → 私密空间互动 → 次日询问饿不饿、送走后保持基本礼貌 → 案例收束。
