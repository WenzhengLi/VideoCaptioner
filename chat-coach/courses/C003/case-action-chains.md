# C003 案例动作链

## 课前答疑：群体局与升级刹车

### 链路 A：原邀约女生被冷落

```text
女生带闺蜜赴约
→ 男生转去和更好看的闺蜜互动
→ 原邀约女生没有被照顾、情绪下头
→ 她说闺蜜可以留下，自己想走
→ 闺蜜观察到她不开心
→ 最终选择陪她一起撤
→ 课程判断男生对两人的吸引和群体照顾都不足
```

### 链路 B：舌吻后突然离开

```text
女生接受喝酒和舌吻
→ 意识到继续可能进入更深升级
→ 欲望与“这样不好”的自我约束同时出现
→ 她起身回家
→ 另一个可能：外形差距或现场互动已耗光好感
→ 男生要根据此前投入和后续回流区分
```

### 链路 C：平台与目标

```text
同一个女生在相亲平台展示可靠长期面
→ 在抖音、社交软件或夜店展示更轻松的一面
→ 渠道给双方不同默认剧本
→ 关系目标要顺着平台和她的表达验证
→ 不与平台默认目的硬拧
```

## 案例一：南宁旧资源琪琪

### 链路 A：旧资源重启

```text
多年未持续联系
→ 用立冬、昵称、电影和天气轻松重开
→ 保持高能量日常交流
→ 女生主动延长话题
→ 男生提出今晚与朋友一起玩
```

### 链路 B：优先级测试与群体邀约

```text
女生：我是备选吧 / 怎么现在才想我
→ 男生解释最近忙、现在终于有时间
→ 邀她进入朋友群体局
→ 女生问还有其他朋友、自己喝不了
→ 男生替她决定地点和安排
→ 她用不给其他小姐姐机会继续暧昧
```

### 链路 C：群体局照顾不足

```text
女生到场并锁定男生
→ 现场女生太多、男生无法照顾
→ 她感觉拥挤、被忽略
→ 默默离开
→ 后续聊天把这次错过变成重新见面的背景
```

### 链路 D：渣男人设与电影切口

```text
女生认为男生是渣男
→ 男生曾试图掰正但失败
→ 课程认为她知道仍继续聊，说明能接受该人设
→ 恐怖片 / 私人影院成为具体切口
→ 男生提出一起看和待在一起
```

### 链路 E：浪漫价值观说服

```text
女生：你了解我吗 / 待在一起有什么意义
→ 男生讲此前相互错过、彼此有感觉
→ 输出不逃避问题、珍惜难得理解
→ 将共同经历和孤独解释为共同痛苦
→ 把再次见面定义为不再错过
→ 女生逐渐进入恋爱幻想
```

### 链路 F：家里顾虑到见面

```text
女生说妈妈不允许随便过夜、第一次去家里害怕
→ 男生解释喜欢的人在一起不叫随便
→ 描绘下雨天一起待、听音乐和陪伴
→ 女生说慢热但对喜欢的人会主动
→ 男生高频输出并电话确认
→ 发位置、线下见面
```

### 链路 G：线下改道与维护

```text
女生精心打扮下楼，提出去咖啡厅
→ 出租车出现
→ 男生直接上车并报自己住处
→ 女生配合改道
→ 线下升温
→ 后续继续调侃、分享和表达感觉
```

## 案例二：低反应到突然行动窗口

### 链路 A：真实性测试与价值观

```text
女生初始反应低
→ 说看你不像缺女生
→ 又问没见过为什么有感觉
→ 男生用照片印象、聊得来和价值观相似回答
→ 继续输出认真恋爱、共同经历和自觉自律
→ 女生逐步增加投入
```

### 链路 B：模糊邀约未接

```text
下雨天
→ 男生描绘一起安静待、听音乐
→ 女生没有接住
→ 课程不把这一次未接直接等同于永远没窗口
→ 继续正常维护
```

### 链路 C：行动窗口突然打开

```text
女生：你又去哪了 / 一直不理我
→ 课程判断她已形成关系预期
→ 女生说喝多、心情不好
→ 男生停止其他安排
→ 打电话、确认位置、开车接人
→ 快速落地相处
```

### 链路 D：商品链接的未来行动

```text
见面后女生发护肤品 / 商品链接
→ 男生不立即用转账证明
→ 也不生硬拒绝
→ 用“过几天去澳门帮你带”转换
→ 既保留回应，又形成未来联系点
```
