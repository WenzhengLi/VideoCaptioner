# C006 案例动作链

## CASE-C006-001：本地大窗口，当天快速落地

```text
女生主动开场并二次追发“好无聊”
→ 讲师判断：主动程度和当前状态共同构成大窗口
→ 男生先镜像双方都空闲，不立刻低位求见
→ 提出“一起待着”的小测试
→ 她惊讶但未拒绝
→ 给出电影、音乐、吃东西、猫和聊天等具体相处画面
→ 她说“也行”并发位置
→ 出现没洗头、没准备、身高和时间不确定
→ 逐项回答具体问题，每解决一项都回到“什么时候来”
→ 约定后继续交换准备、穿搭和兴奋感
→ 连续升温后切回动漫等普通话题形成后撤
→ 到场共享位置、牵手、根据穿搭做定制调侃
```

关键不是背一句邀约，而是把“大窗口—具体场景—现实顾虑—明确时间—到场衔接”连成完整执行链。

## CASE-C006-002：跨城自费来成都

```text
凌晨在线和共同孤独打开长线聊天
→ 用宠物、作息、饮食和日常观察投入与生活可行性
→ 普通话题中穿插轻暧昧，再主动回到日常
→ 用区别性和接纳情绪建立关系想象
→ 她问什么时候回南宁
→ 男生保留暧昧，再把方向转成她来成都
→ 她提出宠物、母亲、时间、陌生城市、被骗和名字等顾虑
→ 逐项缩小成本、降低压力，同时保持邀约框架
→ 她同意最多三天，并在四号/节后之间选择
→ 男生敲定四号，强调“为我而来”
→ 她自费订票，行动窗口落地
→ 临行用生理期和“只为性就退票”确认男生动机
→ 男生继续表达想见并询问照顾需求
→ 到达后应以接机、安排和照顾匹配她的跨城投入
```

这条链说明“没窗口聊到有窗口”不是单点话术，而是连续识别投入、补足信任、转换邀约方向和兑现现实接待。
