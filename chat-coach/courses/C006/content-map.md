# C006 开放式内容地图

状态：完整扫描完成，等待用户确认。

## 一、课程整体在讲什么

C006 用两个速度完全不同的案例说明，窗口不是一个固定标签，而是由主动程度、信息投入、行动成本、顾虑能否被解决以及后续是否继续靠近共同构成。第一个案例是本地大窗口：女生主动开场、再次说“好无聊”，男生迅速把共同空闲转成当天见面，并在洗头、身高、时间等具体问题上逐项解决。第二个案例是跨城慢落地：需要用日常话题、情绪接纳、生活兼容和轻度暧昧连续维持，再把“你什么时候回南宁”转成“你来成都”，最终落实自费机票。

```text
先看主动与投入，判断窗口大小
→ 窗口大时快速给出共同场景和具体安排
→ 顾虑出现时回答顾虑本身，再回到邀约
→ 连续升温后主动回到普通话题，形成推拉
→ 跨城邀约用日常连续性和情绪价值补足信任
→ 将模糊意愿转成天数、日期、票和到达安排
→ 临行测试继续判断她是在确认动机还是退出
→ 对方承担高行动成本后，男生也要匹配接待投入
```

## 二、章节与内容变化

### 02:00—18:00：主动开场、好无聊与当天窗口

- 女生加上后主动说话，男生没及时回，她又发“好无聊呀”；
- 讲师把主动开场和二次追发判断为大窗口；
- 男生先同步“我也没安排”，再提出“一起待着”；
- 她回“啊”但没有拒绝，课程要求继续把安排说具体，而不是因惊讶退掉。

### 18:00—35:00：活动菜单、洗头、身高和时间

- 用电影、音乐、吃东西、猫、聊天和“有感觉的事”组成可想象的相处场景；
- 她说“也行”并给位置，进入执行；
- “没洗头、不确定出不出门”被视为快速私密邀约中应预期的现实顾虑；
- 回答洗头、吹头、护肤用品，再把话题拉回“我等你”；
- 身高问题属于外形筛选，男生用拥抱和下巴靠头顶的画面回答；
- 她给不出时间时，继续要一个明确节点，而不是无限等通知。

### 35:00—48:00：持续上头、主动后撤与到场衔接

- 课程强调短期大窗口要持续传递兴奋、期待和相见画面；
- 但不能一路只升不降，讲师用股票 K 线比喻，升温后切回动漫等普通话题；
- 约定后继续分享准备、穿搭、香味，并问她穿什么；
- 她说“不要这么热情，我不习惯”，课程仍保持高能量但不机械重复同一词；
- 到场发位置、牵手，并根据她的穿搭和主页内容做定制调侃。

### 48:00—65:00：跨城案例的共同孤独与生活可行性

- 女生凌晨仍在线，讲师注意到她的非标准作息；
- “自由的另一个词叫孤寡”建立共同状态；
- 宠物话题不是只聊猫狗，而是看她是否展开、分享并投入；
- 接着询问宠物是否限制旅行，逐步确认她能否来成都；
- 奶茶甜度、吃苦需要甜、健身与饮食等话题建立互补和轻暧昧。

### 65:00—82:00：普通话题、区别性和情绪接纳

- 跨城落地不能只靠连续性暗示，动漫、食物、电视、工作和日常是信任底座；
- “世界上只有两种男人，一种是他们，另一种是我”建立区别性；
- “大事小事，我都可以接受你的情绪和小脾气”被讲师重点肯定；
- 面对修图顾虑，课程判断细小差异不改变整体吸引，更重要的是相处是否匹配；
- 生物钟一致被当作生活兼容信号。

### 82:00—96:00：把回南宁改成来成都

- 女生问什么时候回南宁，男生答“看你想我的程度”，保留暧昧想象；
- 随后转向“你可以来成都看看新城市”；
- 她提出母亲、宠物、停留天数、陌生城市、压力、被拐和连名字都不知道等顾虑；
- 课程逐项缩短停留、说明时间灵活、降低陌生压力，同时保持“不是靠名字喜欢上”的关系框架；
- 顾虑下降后，她同意最多三天；课程认为先让行动发生，再通过现实相处验证兼容。

### 96:00—108:30：敲票、为我而来、临行测试与接待投入

- 她在“国庆后”与“四号”之间犹豫，男生尽快敲定四号；
- 她想提前与朋友同行，男生强调“希望你为我而来，不是顺便来”；
- 她担心独自坐飞机，男生用“整架飞机的人都陪你”偷换概念，轻松化解；
- 订票后仍继续分享和升温，不能只回“好”；
- 她说来生理期、若男生只是为了性就退票，讲师判断这是对动机的确认测试；
- 男生继续邀请，同时转到“怎么缓解、需要什么”以验证和照顾；
- 到达时课程明确指出：对方自费跨城而来，条件允许应当接机，不能把所有行动成本留给她。

## 三、课程核心体系

### 大窗口仍要经过小测试

主动、追发、给位置和接受相处场景说明窗口高，但“啊”“没洗头”“几点不确定”“你多高”仍是执行中的测试。高窗口不是忽略问题，而是有能力逐项回答后继续推进。

### 快落地与慢落地的聊天结构不同

本地当天见面可以快速进入共同场景；跨城则需要普通日常、生活兼容、情绪接纳和轻度暧昧交替，让她愿意承担真实成本。

### 推拉不是故意冷落

课程里的后撤是从连续升温切回动漫、吃饭、宠物和日常，让互动有呼吸；并非在对方已经投入后故意消失或制造焦虑。

### 行动投入需要匹配

她自费买票意味着显著行动投入。课程后段明确承认没有接机做得不好：即使聊天框架强，现实接待也要匹配对方成本。
