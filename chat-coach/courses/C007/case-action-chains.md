# C007 案例动作链

## CASE-C007-001

1. 刚加好友 → 描述自身状态、询问她在做什么 → 女生持续分享位置、疫情和生活信息 → 判断有聊天窗口。
2. 女生说抱玩偶睡 → 顺势描绘“以后互抱入眠” → 她继续接话 → 暧昧升级成立。
3. 她用星座质疑不专一 → 不否认她的话题兴趣，改写为“星座只参考、我会及时止损、对自己人报备” → 价值观对齐。
4. 双方暂时异地 → 日常分享、语音、饮食、工作和奔现预期交替 → 维持熟悉感，不让关系只剩性升级。
5. 解封并可见 → 要求具体周几、几号 → 她给出周末范围 → 确定节点后换话题，不反复逼问。
6. 男生七天没联系 → 女生控诉没想起她 → 解释忙碌但承认落地后没找，表达回来首先想见她 → 重新定义关系。
7. 女生说怕依赖、需要沟通时间 → 讲师强调感觉到位、回应和行动，并指出她当前反应冷淡 → 她继续解释而非退出。
8. 女生提出正式约会 → 用“公司离不开，但你来就有理由走”把她接入自己的生活 → 她同意来找。
9. 月经和身体不适 → 提供暖宝宝、休息、照顾和用品 → 状态恢复后立即重启邀约。
10. 女生答应来并谈睡衣 → 明确接送、到达时间和过夜准备 → 行动窗口成形。
11. 临出门担心晚和喝多 → 先表达等待与期盼，再给不方便可理解的退路 → 她澄清没有其他事、尽量早点结束。
12. 发地址、到达、见面 → 线下落地 → 次日继续关心身体状态，完成案例闭环。
