# C017 案例动作链

## CASE-C017-001：临行前夜从线上拉扯到凌晨见面

1. 女生夸帅 → 男生奖励她会聊天但延迟数小时 → 女生降温。
2. 男生继续提供有限话题并问是否方便 → 女生给出在和朋友聊天的客观原因 → 男生结束当晚。
3. 女生次日主动重开 → 交换位置、十五分钟距离、作息与单身状态 → 短期窗口形成。
4. 女生用“孤单、想安定”表达需求 → 男生接“在一起就不孤单” → 暧昧加速。
5. 女生说不找艳遇、自己普通、太快不考虑 → 男生接她仍开放的“可以认识”，同时表达聊得来、有感觉和有标准。
6. 女生说晚、不想跑 → 男生提出接她；说不想收拾 → 男生说不用收拾、自己也没打理。
7. 连续阻力到第三轮 → 男生说明次日离开并准备撤 → 女生追问离开多久，窗口回流。
8. 女生担心对谁都这样、素颜不好看 → 男生表达不是每个人、有标准、聊得来更重要 → 女生同意见一下。
9. 男生十三分钟到楼下接人 → 现场聊天后带回家 → 私密互动没有形成良好体验。
10. 女生提出回去并独自离开 → 男生睡着未指路 → 讲师事后指出应送到楼下完成收尾。
