# C014 案例动作链

## CASE-C014-001：群内战绩浅看

```text
多线同时出现行动窗口
→ 需要判断谁已进入具体行动
→ 分别安排吃饭/到家/现场活动
→ 信息只作背景，不扩写成完整课程事实
```

## CASE-C014-002：广西女生六天异地订票

```text
女生迎合并评价地域男性
→ 男生点破不舒服的标签，表达自己的框架
→ 女生核验条件与社交软件状态
→ 表达兴趣和专一感
→ 先判断近期是否空闲
→ 表达想待一起，同时关心生病症状和药
→ 节日孤独进入恋爱意愿
→ 前任/失恋/长久安全题
→ 深读朋友圈，以艺术和生活兴趣表达看见
→ “相处不是草率、长久靠维护”
→ 见面比网络了解更真实
→ 她反复确认一时兴起、身份、遇事处理
→ 男生重复确认长期可能性
→ 她换星座/分享话题，安全题过关
→ 分享、依赖、日常同频积累好感
→ 踩油门到亲密场景
→ 她说慌、太快、行动不便
→ 接机、住处、日期等现实解法
→ 她测试官宣和注销软件
→ 给态度并继续锁日期
→ 几天失联引发谁先联系的拉扯
→ 说清不喜欢突然消失
→ 回到三十号、待几天和票务
→ 航班确定，等待接机
```
