# C013 案例动作链

## CASE-C013-001：低窗口到见面、过夜与关系定位

```text
平淡回复且对展示面存疑
→ 确认距离，观察她主动问发型的外形兴趣
→ 她表达对当地男性的负面体验
→ 先问经历、顺应感受，不立刻反驳
→ 询问隔天没事的休息时间，含蓄测试长时间相处
→ 她连续追问“待一起/有感觉是什么意思”
→ 男生用甜蜜相处、双方标准和现场再判断点到为止
→ 她继续哈哈和追问，讲师判为接住框架
→ 三十七分钟视频确认安全、价值、聊天舒适和时间
→ 第一次安排被男生延后，互动放凉
→ 第二轮重新补安全与温度
→ 用吃零食、变胖、床边、睡醒等情侣日常描绘
→ 她反应提高后才使用爱称
→ 出差形成短暂稀缺，提出今晚相处和次日早餐
→ 她发电话、出门并讨论用品
→ 男生楼下接，牵手拥抱，先在客厅熟悉环境
→ 现场升级并过夜
→ 女生要求明确认真或短期关系
→ 男生必须补上身份与真实意图
```

## CASE-C013-002：平安夜群内约出与现场恢复

```text
多名学员同时约会并在群内汇报
→ 女生提出“太快，想慢慢发展”
→ 不只看这句话，继续读取她是否在场、是否持续对话
→ 通过具体现场语言和下一步安排降低速度压力
→ 她接受进入休息/继续相处
→ 次日复盘关系、体验和是否需要结束联系
```

第二案材料以展示和碎片求助为主，不足以支持大量独立课程事实；正式候选应以第一案为主，第二案只补“太快”不能脱离行动判断。
