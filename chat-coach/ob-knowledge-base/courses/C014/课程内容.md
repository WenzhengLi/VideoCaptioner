# C014 课程内容

主案例从异地匹配开始。男生先处理地域评价和专一感，再判断女生近期是否有空；对方生病时先解决症状，而不是借照顾强冲。节日孤独、恋爱意愿、前任和朋友圈深读建立长期感；女生随后反复测试太快、一时兴起、明确身份、官宣和软件注销。男生一面保留现实见面再了解，一面给出长期可能性和遇事解决的态度。她换题后进入星座、分享、依赖和亲密升级；最后以接机、待几天、日期和航班完成异地订票。

课程强调：安全题答到点后对方会换题；“慌、太快”必须与她是否继续讨论现实行动一起判断；好感营造与关系升级交替；情绪拉扯后回到执行。
