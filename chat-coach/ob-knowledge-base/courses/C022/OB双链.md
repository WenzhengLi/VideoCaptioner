# C022 OB 双链

以下均为候选索引，完整四方向、多示例和后续分支见[候选补足](../../../courses/C022/supplement-review.md)。

## <a id="ob-c022-001"></a>OB-C022-001：展示面升级后回复上升
- 标签：[吸引建立](../../课程场景标签.md#tag-stage-attraction)、[窗口上升](../../课程场景标签.md#tag-signal-window-up)、[分享](../../课程场景标签.md#tag-action-sharing)

- supplement_chain：真实片段 → 反向了解 → 坏坏带入 → 轻邀约。

## <a id="ob-c022-002"></a>OB-C022-002：日常分享确认共同生活世界
- 标签：[分享场景](../../课程场景标签.md#tag-scene-sharing)、[未来画面](../../课程场景标签.md#tag-scene-future-picture)、[场景描绘](../../课程场景标签.md#tag-action-scene-painting)
- course_chain：烟花/旅行分享 → 共同偏好 → 双人画面 → 未来共同体验。
- supplement_chain：共鸣细节 → 让她描绘 → 坏坏同盟 → 未来落点。

## <a id="ob-c022-003"></a>OB-C022-003：“是不是对所有女生都这样”
- 标签：[真实性测试](../../课程场景标签.md#tag-scene-sincerity-test)、[认真程度](../../课程场景标签.md#tag-psych-seriousness)、[特殊性](../../课程场景标签.md#tag-psych-specialness)
- course_chain：她核验花心/短期 → 重构经历与滥情 → 提供观察空间 → 回到相处。
- supplement_chain：简洁区分 → 给观察权 → 坏坏反问 → 行动验证。

## <a id="ob-c022-004"></a>OB-C022-004：异地与关系结果顾虑
- 标签：[需要安全感](../../课程场景标签.md#tag-psych-safety)、[提供筹码](../../课程场景标签.md#tag-action-chips)、[有兴趣但犹豫](../../课程场景标签.md#tag-psych-interested-hesitant)
- course_chain：异地成本 → 她犹豫投入 → 承认现实 → 补可预期性。
- supplement_chain：承认现实 → 小步验证 → 行程筹码 → 坏坏坚定。

## <a id="ob-c022-005"></a>OB-C022-005：邀约被略过后的海浪回收
- 标签：[邀约阶段](../../课程场景标签.md#tag-stage-invitation)、[后撤](../../课程场景标签.md#tag-action-withdraw)、[推拉](../../课程场景标签.md#tag-action-push-pull)
- course_chain：她略过邀约但继续聊天 → 判断非明确拒绝 → 回收压力 → 等新信号再推。
- supplement_chain：顺她话题 → 轻点到为止 → 补熟悉感 → 新信号再邀。

## <a id="ob-c022-006"></a>OB-C022-006：吸引后补长期可预期性
- 标签：[需要安全感](../../课程场景标签.md#tag-psych-safety)、[主动报备](../../课程场景标签.md#tag-signal-active-report)、[轻占有](../../课程场景标签.md#tag-action-light-possession)

- supplement_chain：真实节奏 → 报备预期 → 坏坏稳定 → 共同安排。

## <a id="ob-c022-007"></a>OB-C022-007：共同内容转未来画面
- 标签：[未来画面](../../课程场景标签.md#tag-scene-future-picture)、[场景描绘](../../课程场景标签.md#tag-action-scene-painting)、[浪漫](../../课程场景标签.md#tag-tone-romantic)

- supplement_chain：轻画面 → 双人画面 → 坏坏占位 → 具体计划。

## <a id="ob-c022-008"></a>OB-C022-008：模糊想见落到执行
- 标签：[邀约落地](../../课程场景标签.md#tag-stage-invitation-landing)、[邀约内容确认](../../课程场景标签.md#tag-scene-invite-content)、[行动配合](../../课程场景标签.md#tag-signal-action-cooperation)
