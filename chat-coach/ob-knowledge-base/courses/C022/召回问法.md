# C022 召回问法

- `OB-C022-001`：她看了朋友圈开始主动问；展示面升级后怎么接；她突然回复变多。
- `OB-C022-002`：她说也喜欢烟花；怎么把旅行聊成共同体验；日常分享有什么用。
- `OB-C022-003`：她问你是不是对谁都这样；她说你很花；怎么证明不是短期。
- `OB-C022-004`：她说异地没结果；不在一个城市怎么聊；她怕投入没结果。
- `OB-C022-005`：她没接邀约但还聊天；邀约被绕开；推进后怎么回收。
- `OB-C022-006`：她问你是不是总忙；怎么给长期安全感；怎么稳定联系。
- `OB-C022-007`：她说一直想去；怎么描绘以后一起；共同兴趣怎么升温。
- `OB-C022-008`：她说回来再见；有机会见是什么意思；怎么把模糊邀约定下来。
