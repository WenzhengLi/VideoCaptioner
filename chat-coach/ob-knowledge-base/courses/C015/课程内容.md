# C015 课程内容

本课有两个完整案例。第一案训练从现场语音、旁人、时间和女生动作读出“求救/脱身/接走”信号，再以快速确认位置、照顾气氛和实际接人完成截胡。第二案训练高窗口识别：主动、回复量、问题量、共鸣、夸奖和行动叠加后，及时邀约；面对“不随便、慢慢相处、不急于肢体、一夜情、炮友”反复顾虑，尊重她的人设，表达心灵、关系可能、真实吸引和责任，同时不让互动无限停在网上。最后通过机场前时间重算、早餐场景、位置、电话和叫车完成当晚落地。

课程后段补充：线上安全与长期感必须在线下继续兑现；环境、照顾、关系意图和健康顾虑未解决时，只顾肢体会导致抗拒。明确拒绝应停，不得把“不要”机械理解为“要”。
