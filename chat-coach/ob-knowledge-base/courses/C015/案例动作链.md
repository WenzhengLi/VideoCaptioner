# C015 案例动作链

## CASE-C015-001
```text
旁边男人语音 → 判断她想脱身/被缠 → 问情况与位置 → 提出去接
→ 口头不用但继续回答 → 三次问去哪 → 解释照顾与陪伴气氛
→ 接受安排 → 到场接走
```

## CASE-C015-002
```text
主动/表情/问题/共鸣/夸帅 → 高窗口 → 事业与标准人设 → 及时邀约
→ 不随便/慢慢来/不急肢体 → 尊重人设 + 心灵优先 + 可无动作见面
→ 一夜情/炮友反复确认 → 欲望、责任、长期可能并存
→ 明晚机场/回来再见 → 重算今晚窗口 + 早餐 + 不错过
→ 位置/电话/准备/下楼/叫车 → 到场过夜
→ 继续维持温柔照顾，明确拒绝则停止
```
