## <a id="ob-c012-001"></a>OB-C012-001：第一天聊一段后养鱼
- 标签：[真实性测试](../../课程场景标签.md#tag-scene-sincerity-test)、[需要安全感](../../课程场景标签.md#tag-psych-safety)、[推拉](../../课程场景标签.md#tag-action-push-pull)、[坏坏](../../课程场景标签.md#tag-tone-bad)
- course_chain：`第一天聊了一段，然后断了。` → 课程认为加人后先快速判断窗口；有窗口继续约，没有窗口就放入朋友圈观察，而不是长期陪聊。 → 低成本完成身份、感情方向和安全感三个基本信息，再停手观察。
- supplement_chain：需要补足什么时候继续、什么时候删除和怎么重启。 → 建立低成本筛选而非机械群聊 → 四种策略与后续分支。
- 状态：`pending_user_confirmation`。

## <a id="ob-c012-002"></a>OB-C012-002：八月拍一拍重新开窗
- 标签：[未来场景](../../课程场景标签.md#tag-scene-future-picture)、[邀约](../../课程场景标签.md#tag-action-invite)、[窗口](../../课程场景标签.md#tag-concept-window)、[直球](../../课程场景标签.md#tag-action-direct)
- course_chain：`她拍了拍我。` → 课程把沉默数月后的拍一拍视为低成本重启信号，男方用‘怎么说’让她承担后续内容。 → 轻回应，不替她编话题，看她是否继续解释。
- supplement_chain：需补足误触、试探和真想联系的识别。 → 让主动方继续投入 → 四种策略与后续分支。
- 状态：`pending_user_confirmation`。

## <a id="ob-c012-003"></a>OB-C012-003：你在成都都不找我
- 标签：[窗口](../../课程场景标签.md#tag-concept-window)、[邀约](../../课程场景标签.md#tag-action-invite)、[直球](../../课程场景标签.md#tag-action-direct)、[坏坏](../../课程场景标签.md#tag-tone-bad)
- course_chain：`你在成都，你都不找我。` → 课程认为这是明确行动窗口：她不仅关注位置，还对未被邀约表达不满。 → 承认前面时间没对上，立刻问空档并转向邀约。
- supplement_chain：需补足真想见、索取关注和单纯抱怨三类。 → 把抱怨识别为可能的邀约接口 → 四种策略与后续分支。
- 状态：`pending_user_confirmation`。

## <a id="ob-c012-004"></a>OB-C012-004：爱情不是想要就立刻拥有
- 标签：[真实性测试](../../课程场景标签.md#tag-scene-sincerity-test)、[需要安全感](../../课程场景标签.md#tag-psych-safety)、[推拉](../../课程场景标签.md#tag-action-push-pull)、[坏坏](../../课程场景标签.md#tag-tone-bad)
- course_chain：`没有爱情就只有赚钱了。` → 课程回复‘都不耽误’，把事业和感情从二选一框架里拆开。 → 认同赚钱现实，同时保留关系窗口。
- supplement_chain：需补足忙碌、失望和防御性独立三种动因。 → 解除事业与感情的假性对立 → 四种策略与后续分支。
- 状态：`pending_user_confirmation`。

## <a id="ob-c012-005"></a>OB-C012-005：你条件这么好应该不缺
- 标签：[未来场景](../../课程场景标签.md#tag-scene-future-picture)、[邀约](../../课程场景标签.md#tag-action-invite)、[窗口](../../课程场景标签.md#tag-concept-window)、[直球](../../课程场景标签.md#tag-action-direct)
- course_chain：`你条件这么好，应该不缺。` → 她在确认男方是否单身、是否多线、自己是否特殊；课程解释上一段感情和工作填满生活。 → 不急着炫耀选择，用具体经历说明当前状态。
- supplement_chain：需要补足特殊性、反筛选和诚实边界。 → 回答竞争焦虑并保留高位 → 四种策略与后续分支。
- 状态：`pending_user_confirmation`。

## <a id="ob-c012-006"></a>OB-C012-006：我要找结婚的不想耗
- 标签：[窗口](../../课程场景标签.md#tag-concept-window)、[邀约](../../课程场景标签.md#tag-action-invite)、[直球](../../课程场景标签.md#tag-action-direct)、[坏坏](../../课程场景标签.md#tag-tone-bad)
- course_chain：`我要找一个结婚的，不想谈很久对方却不结。` → 她表达年龄焦虑和结果需求；课程直接满足需求，说自己也想找靠谱能走到最后的人。 → 先认同时间成本，再区分认真方向和提前保证。
- supplement_chain：缺少对结婚期限、价值观和相处验证的进一步提问。 → 对齐长期目标但不画饼 → 四种策略与后续分支。
- 状态：`pending_user_confirmation`。

## <a id="ob-c012-007"></a>OB-C012-007：等解封甜甜在一起

- supplement_chain：需补足不同亲密强度和公共/私密地点选择。 → 用场景描绘推动见面想象 → 四种策略与后续分支。
- 状态：`pending_user_confirmation`。

## <a id="ob-c012-008"></a>OB-C012-008：在家不去干嘛的明牌窗口
- 标签：[未来场景](../../课程场景标签.md#tag-scene-future-picture)、[邀约](../../课程场景标签.md#tag-action-invite)、[窗口](../../课程场景标签.md#tag-concept-window)、[直球](../../课程场景标签.md#tag-action-direct)
- course_chain：`在家，不去干嘛。` → 课程认为语气和表情已经在给邀约接口，后续用统一邀约内容并很快发位置见面。 → 直接邀请来待着，或过去接她，不再绕无关话题。
- supplement_chain：需补足独处邀请、公共见和只想聊天三种梯度。 → 及时承接行动窗口 → 四种策略与后续分支。
- 状态：`pending_user_confirmation`。

## <a id="ob-c012-009"></a>OB-C012-009：快速发位置与十二分钟落地
- 标签：[窗口](../../课程场景标签.md#tag-concept-window)、[邀约](../../课程场景标签.md#tag-action-invite)、[直球](../../课程场景标签.md#tag-action-direct)、[坏坏](../../课程场景标签.md#tag-tone-bad)
- course_chain：`你发位置，我过去接你。` → 课程展示窗口明确后从聊天迅速转为路线、准备和到楼下，不继续制造新的文字障碍。 → 确认地址、预计时间和见面动作。
- supplement_chain：需补足临时变化、迟到和安全确认。 → 把高窗口转成稳定执行 → 四种策略与后续分支。
- 状态：`pending_user_confirmation`。

## <a id="ob-c012-010"></a>OB-C012-010：约会不是去酒店的前戏
- 标签：[真实性测试](../../课程场景标签.md#tag-scene-sincerity-test)、[需要安全感](../../课程场景标签.md#tag-psych-safety)、[推拉](../../课程场景标签.md#tag-action-push-pull)、[坏坏](../../课程场景标签.md#tag-tone-bad)
- course_chain：`吃饭时怎么给性暗示，怎么带去酒店？` → 讲师纠正这种目标锁定：吃饭就是相互了解和展示魅力，满脑子酒店会让现场下头。 → 先把状态、节奏、真诚和表达好感做好，再看真实窗口。
- supplement_chain：需补足从舒适到关系升级的动作链。 → 把结果焦虑改成现场体验 → 四种策略与后续分支。
- 状态：`pending_user_confirmation`。

## <a id="ob-c012-011"></a>OB-C012-011：精致女生让男生自卑
- 标签：[未来场景](../../课程场景标签.md#tag-scene-future-picture)、[邀约](../../课程场景标签.md#tag-action-invite)、[窗口](../../课程场景标签.md#tag-concept-window)、[直球](../../课程场景标签.md#tag-action-direct)
- course_chain：`她朋友圈很精致，我不敢聊，还想长期。` → 课程给两条路：真正提升到同等级，或暂时放下自卑只看她的真实反应；长期拥有更要求生活层级匹配。 → 先区分想认识、短期约会和长期关系的不同成本。
- supplement_chain：需补足不靠伪装的价值呈现和筛选。 → 从仰望转向匹配判断 → 四种策略与后续分支。
- 状态：`pending_user_confirmation`。
