# C006 OB 双链

状态：候选索引。完整证据、方向和回复见 [C006 候选补足稿](../../../courses/C006/supplement-review.md)。

## <a id="ob-c006-001"></a>OB-C006-001：主动追发“好无聊”
- 标签：[破冰](../../课程场景标签.md#tag-stage-break-ice)、[状态分享](../../课程场景标签.md#tag-scene-sharing)、[窗口上升](../../课程场景标签.md#tag-signal-window-up)、[邀约](../../课程场景标签.md#tag-stage-invitation)
- `course_chain`：主动开场 → 未获回复再说无聊 → 讲师判为大窗口 → 同步双方空闲 → 进入共同场景。
- `supplement_chain`：状态分享/索取陪伴/明确想见 → 共振、点破、活动或测投入 → 依位置与行动继续。

## <a id="ob-c006-002"></a>OB-C006-002：大窗口下保持平行状态
- 标签：[窗口上升](../../课程场景标签.md#tag-signal-window-up)、[快速邀约](../../课程场景标签.md#tag-scene-fast-invite)、[邀约](../../课程场景标签.md#tag-stage-invitation)
- `course_chain`：她说没安排 → 男生不低位求见 → 先表达自己也空闲 → 提出一起待着。
- `supplement_chain`：平行同步 → 轻占用/给选择/直接推进 → 同意、犹豫或拒绝分别处理。

## <a id="ob-c006-003"></a>OB-C006-003：“啊”后的含义核验
- 标签：[黄灯](../../课程场景标签.md#tag-signal-yellow)、[持续回复](../../课程场景标签.md#tag-signal-continuous-reply)、[有兴趣但犹豫](../../课程场景标签.md#tag-psych-interested-hesitant)
- `course_chain`：提出一起待着 → 她回“啊”但未拒绝 → 讲师判惊讶/害羞 → 男生说具体活动。
- `supplement_chain`：惊讶/没听懂/不想见 → 解释、调侃、降级或核验 → 用后续投入确认。

## <a id="ob-c006-004"></a>OB-C006-004：活动菜单与相处画面
- 标签：[邀约内容确认](../../课程场景标签.md#tag-scene-invite-content)、[未来场景](../../课程场景标签.md#tag-scene-future-picture)、[私密地点邀约](../../课程场景标签.md#tag-scene-private-invite)
- `course_chain`：她问做什么 → 给电影、音乐、吃东西、猫和聊天 → 她说也行并给位置。
- `supplement_chain`：窗口强弱 → 轻日常/共同体验/坏坏留白/她主导 → 落到活动与地点。

## <a id="ob-c006-005"></a>OB-C006-005：没洗头与准备顾虑
- 标签：[快速邀约](../../课程场景标签.md#tag-scene-fast-invite)、[黄灯](../../课程场景标签.md#tag-signal-yellow)、[行动配合](../../课程场景标签.md#tag-signal-action-cooperation)
- `course_chain`：已给位置 → 说没洗头不确定出门 → 解准备问题 → 回到“我等你”。
- `supplement_chain`：形象压力/准备麻烦/礼貌退出 → 减压、提供条件、调侃或核验 → 锁时间或停止。

## <a id="ob-c006-006"></a>OB-C006-006：身高与外形筛选
- 标签：[真实性测试](../../课程场景标签.md#tag-scene-sincerity-test)、[建立吸引](../../课程场景标签.md#tag-stage-attraction)、[未来场景](../../课程场景标签.md#tag-scene-future-picture)
- `course_chain`：执行前问身高 → 讲师判身体筛选 → 报数字并描绘拥抱站位 → 继续约定。
- `supplement_chain`：真实条件/暧昧画面/反测/不匹配 → 四种回答 → 回到行动或尊重筛选。

## <a id="ob-c006-007"></a>OB-C006-007：不给具体时间
- 标签：[模糊延期](../../课程场景标签.md#tag-scene-vague-delay)、[邀约落地](../../课程场景标签.md#tag-stage-invitation-landing)、[黄灯](../../课程场景标签.md#tag-signal-yellow)
- `course_chain`：方向同意但时间模糊 → 不无限等待通知 → 要具体时间或确认节点。
- `supplement_chain`：日程未定/行动犹豫 → 锁时间、锁节点、轻推动或给退出 → 到点执行。

## <a id="ob-c006-008"></a>OB-C006-008：升温后的主动后撤
- 标签：[暧昧升温](../../课程场景标签.md#tag-stage-flirting)、[状态分享](../../课程场景标签.md#tag-scene-sharing)、[建立吸引](../../课程场景标签.md#tag-stage-attraction)
- `course_chain`：连续性张力 → 讲师用 K 线解释不能直线上升 → 切动漫日常 → 再择机回升。
- `supplement_chain`：兴奋重复/压力上升 → 兴趣、生活、推拉或留悬念 → 她主动回升或安心执行。

## <a id="ob-c006-009"></a>OB-C006-009：约定后的共同期待
- 标签：[邀约落地](../../课程场景标签.md#tag-stage-invitation-landing)、[行动配合](../../课程场景标签.md#tag-signal-action-cooperation)、[未来场景](../../课程场景标签.md#tag-scene-future-picture)
- `course_chain`：时间已定 → 双方分享准备、穿搭和香味 → 保持积极情绪 → 出发。
- `supplement_chain`：期待/紧张/延迟 → 准备同步、穿搭互动、轻占有或降压 → 确认出发。

## <a id="ob-c006-010"></a>OB-C006-010：不习惯你的热情
- 标签：[黄灯](../../课程场景标签.md#tag-signal-yellow)、[有兴趣但犹豫](../../课程场景标签.md#tag-psych-interested-hesitant)、[暧昧升温](../../课程场景标签.md#tag-stage-flirting)
- `course_chain`：她已同意见面 → 说不习惯高热情 → 男生不突然冷掉 → 换表达并继续执行。
- `supplement_chain`：害羞/节奏快/真实不适 → 降半档、保持、核验或转执行 → 以语气和行动判断。

## <a id="ob-c006-011"></a>OB-C006-011：自由与共同孤独
- 标签：[状态分享](../../课程场景标签.md#tag-scene-sharing)、[破冰](../../课程场景标签.md#tag-stage-break-ice)、[建立吸引](../../课程场景标签.md#tag-stage-attraction)
- `course_chain`：她说一个人自由 → 男生转为“另一个词叫孤寡” → 建立共同状态 → 进入陪伴画面。
- `supplement_chain`：享受独处/深夜孤单 → 认同、共鸣、占位或核验 → 决定是否增加联系。

## <a id="ob-c006-012"></a>OB-C006-012：宠物投入与旅行可行性
- 标签：[状态分享](../../课程场景标签.md#tag-scene-sharing)、[持续回复](../../课程场景标签.md#tag-signal-continuous-reply)、[邀约](../../课程场景标签.md#tag-stage-invitation)
- `course_chain`：展开宠物细节 → 讲师观察投入 → 问是否限制出行 → 确认家人/寄养可解决。
- `supplement_chain`：兴趣分享/生活约束 → 奖励、兼容、邀约测试或调侃 → 判断跨城条件。

## <a id="ob-c006-013"></a>OB-C006-013：普通日常支撑跨城信任
- 标签：[状态分享](../../课程场景标签.md#tag-scene-sharing)、[建立吸引](../../课程场景标签.md#tag-stage-attraction)、[未来场景](../../课程场景标签.md#tag-scene-future-picture)
- `course_chain`：日常话题 → 穿插轻暧昧 → 主动撤回日常 → 连续性和兼容感上升。
- `supplement_chain`：跨城高成本 → 共同日常、生活兼容、日常埋暧昧或留白 → 稳定投入后谈日期。

## <a id="ob-c006-014"></a>OB-C006-014：接住情绪和小脾气
- 标签：[需要安全感](../../课程场景标签.md#tag-psych-safety)、[确认特殊性](../../课程场景标签.md#tag-psych-specialness)、[关系升级](../../课程场景标签.md#tag-stage-escalation)
- `course_chain`：她暴露情绪顾虑 → 男生说能接受大事小事和小脾气 → 讲师重点肯定 → 区别性增强。
- `supplement_chain`：脆弱/需要哄/沟通习惯 → 稳定、具体承接、坏坏框架或双方边界 → 进入真实事件验证。

## <a id="ob-c006-015"></a>OB-C006-015：修图与真人差异
- 标签：[真实性测试](../../课程场景标签.md#tag-scene-sincerity-test)、[需要安全感](../../课程场景标签.md#tag-psych-safety)、[建立吸引](../../课程场景标签.md#tag-stage-attraction)
- `course_chain`：她担心照片修过 → 课程认为小差异不改整体吸引 → 转向现场兼容 → 继续见面。
- `supplement_chain`：普通修图/明显信息差 → 减压、核验、兼容框架或调侃 → 视频或真实照片确认。

## <a id="ob-c006-016"></a>OB-C006-016：从回南宁转成来成都
- 标签：[邀约](../../课程场景标签.md#tag-stage-invitation)、[未来场景](../../课程场景标签.md#tag-scene-future-picture)、[窗口上升](../../课程场景标签.md#tag-signal-window-up)
- `course_chain`：她问何时回南宁 → “看你想我的程度” → 保留暧昧 → 提出她来成都。
- `supplement_chain`：想见接口/行动未知 → 点窗口、画面、小步测试或双向投入 → 落天数日期。

## <a id="ob-c006-017"></a>OB-C006-017：陌生城市与安全信息
- 标签：[需要安全感](../../课程场景标签.md#tag-psych-safety)、[有兴趣但犹豫](../../课程场景标签.md#tag-psych-interested-hesitant)、[黄灯](../../课程场景标签.md#tag-signal-yellow)
- `course_chain`：她提陌生、被拐、名字未知 → 缩短天数和降低压力 → 保持关系框架 → 同意最多三天。
- `supplement_chain`：真实安全顾虑 → 可验证信息、低成本方案、浪漫框架或轻化解 → 顾虑下降才出票。

## <a id="ob-c006-018"></a>OB-C006-018：自费订票与专程意义
- 标签：[行动配合](../../课程场景标签.md#tag-signal-action-cooperation)、[确认特殊性](../../课程场景标签.md#tag-psych-specialness)、[邀约落地](../../课程场景标签.md#tag-stage-invitation-landing)
- `course_chain`：她准备自费来 → 想和朋友同行顺便见 → 男生强调“为我而来” → 敲定日期出票。
- `supplement_chain`：专程/同行安全/行动奖励 → 特殊性、兼容同行、奖励或调侃 → 以出票验证。

## <a id="ob-c006-019"></a>OB-C006-019：生理期与动机测试
- 标签：[关系确认测试](../../课程场景标签.md#tag-scene-relationship-test)、[确认认真程度](../../课程场景标签.md#tag-psych-seriousness)、[需要安全感](../../课程场景标签.md#tag-psych-safety)
- `course_chain`：订票后说生理期 → “只为性就退票” → 课程判动机测试 → 继续想见并询问缓解。
- `supplement_chain`：确认见人/身体不适/仍有暧昧 → 明确动机、具体照顾、保留吸引或轻核验 → 调整行程。

## <a id="ob-c006-020"></a>OB-C006-020：跨城到达后的接机投入
- 标签：[邀约落地](../../课程场景标签.md#tag-stage-invitation-landing)、[行动配合](../../课程场景标签.md#tag-signal-action-cooperation)、[见面](../../课程场景标签.md#tag-stage-meeting)
- `course_chain`：她自费跨城到达 → 男生没接机 → 讲师明确复盘做得不好 → 条件允许应匹配接待投入。
- `supplement_chain`：能接/确有冲突/她已生气 → 直接接、完整替代方案、奖励投入或迎接 → 休息与后续安排。
