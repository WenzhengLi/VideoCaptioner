# C029 OB 双链

## <a id="ob-c029-001"></a>OB-C029-001：临门带狗或闺蜜
- 标签：[服从性测试](../../课程场景标签.md#tag-concept-compliance-test)、[框架](../../课程场景标签.md#tag-concept-frame)、[给选择](../../课程场景标签.md#tag-action-choice)
- course_chain：一对一已定 → 临时加人/宠物 → 判断顾虑或测试 → 拒绝或改天。
- supplement_chain：问原因 → 保持一对一 → 安全替代 → 坏坏框架。

## <a id="ob-c029-002"></a>OB-C029-002：加了不说话的主动窗口
- 标签：[主动追问](../../课程场景标签.md#tag-signal-active-question)、[窗口上升](../../课程场景标签.md#tag-signal-window-up)、[快速建立吸引](../../课程场景标签.md#tag-stage-fast-attraction)
- course_chain：她主动催聊 → 接住窗口 → 位置与距离 → 快速推进。
- supplement_chain：轻松承接 → 状态切入 → 位置拉近 → 坏坏奖励。

## <a id="ob-c029-003"></a>OB-C029-003：不缺女朋友吧
- 标签：[价值测试](../../课程场景标签.md#tag-scene-value-test)、[高位可得性](../../课程场景标签.md#tag-concept-high-availability)、[表达特殊性](../../课程场景标签.md#tag-action-specialness)
- course_chain：高价值预期 → 不低位自证 → 关系选择标准 → 保留可得性。
- supplement_chain：承认社交 → 选择标准 → 特殊性 → 坏坏反测试。

## <a id="ob-c029-004"></a>OB-C029-004：生日分手与被辜负
- 标签：[长消息](../../课程场景标签.md#tag-signal-long-message)、[需求识别](../../课程场景标签.md#tag-action-need-recognition)、[温柔](../../课程场景标签.md#tag-tone-gentle)
- course_chain：前任伤痛 → 接情绪 → 看见付出 → 展示自己的照顾方式。
- supplement_chain：共情事件 → 看见付出 → 关系观 → 拉回当下。

## <a id="ob-c029-005"></a>OB-C029-005：仪式感与付出测试
- 标签：[付出测试](../../课程场景标签.md#tag-scene-investment-test)、[价值筛选](../../课程场景标签.md#tag-stage-value-screening)、[价值观植入](../../课程场景标签.md#tag-action-value-injection)
- course_chain：询问礼物/零花钱 → 愿意投入但看关系价值 → 不设无条件标准。
- supplement_chain：愿意投入 → 看双方 → 非金钱筹码 → 坏坏筛选。

## <a id="ob-c029-006"></a>OB-C029-006：线上关系意向不等于线下既定关系
- 标签：[首次见面](../../课程场景标签.md#tag-scene-first-meeting)、[关系测试](../../课程场景标签.md#tag-scene-relationship-test)、[线下重新评估](../../课程场景标签.md#tag-psych-offline-reevaluation)
- course_chain：线上默认 → 男生当事实 → 女生只当意向 → 线下重新评估。
- supplement_chain：见前校准 → 轻松降压 → 线下观察 → 给特殊性不抢关系。

## <a id="ob-c029-007"></a>OB-C029-007：高铁站突发与牵手失误
- 标签：[现场决策](../../课程场景标签.md#tag-action-live-decision)、[见面](../../课程场景标签.md#tag-stage-meeting)、[后撤](../../课程场景标签.md#tag-action-withdraw)
- course_chain：同事在场 → 贸然出现 → 急牵手 → 她生气 → 男生退缩离场。
- supplement_chain：远处协调 → 绅士带领 → 失误修复 → 不逃避收尾。
