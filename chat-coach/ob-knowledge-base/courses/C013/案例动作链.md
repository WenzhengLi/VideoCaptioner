# C013 案例动作链

## CASE-C013-001

```text
平淡回复与摆拍疑虑
→ 距离/发型兴趣/经历倾听建立熟悉
→ 含蓄询问完整休息时间
→ 她连续追问男女意图
→ 甜蜜相处 + 双方标准 + 现场判断
→ 视频确认安全、价值、舒适、时间
→ 首次安排被延后
→ 第二轮重新加热
→ 情侣日常 + 适时爱称 + 出差稀缺 + 早餐
→ 电话、用品和到场
→ 楼下接、牵手拥抱、客厅熟悉
→ 现场升级
→ 她要求明确身份
→ 补上真实关系意图
```

## CASE-C013-002

```text
女生说太快、想慢慢发展
→ 不只读单句
→ 结合她是否继续在场/对话/接受下一步
→ 明确拒绝则停止，仍互动则拆小动作继续判断
```
