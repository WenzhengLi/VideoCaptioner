# C013 召回问法

- 她刚加我只回“666”“可以”，是不是没戏？
- 她说我照片像摆拍，怎么接？
- 她主动问我现在发型和头像一样吗，什么意思？
- 她说四川男人都很下头，我要反驳吗？
- 怎么含蓄问她哪天可以待久一点？
- 她一直问“待一起是什么意思”，要不要说破？
- 视频通话到底要确认什么？
- 已经约好了但我临时改期，怎么重新加热？
- 怎么描绘情侣日常，而不是只会聊骚？
- 什么时候可以叫宝宝、乖乖？
- 我要出差了，怎么用时间窗口推进见面？
- 她到楼下后怎么接、怎么让她先放松？
- 她说太快、想慢慢来，还能继续吗？
- 发生关系后她问我们是什么关系，怎么回答？
- 事后她突然冷淡，是不是因为没给身份？
