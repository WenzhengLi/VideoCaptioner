# C025 OB 双链

## <a id="ob-c025-001"></a>OB-C025-001：自然展示面承接聊天
- 标签：[吸引建立](../../课程场景标签.md#tag-stage-attraction)、[分享场景](../../课程场景标签.md#tag-scene-sharing)、[分享](../../课程场景标签.md#tag-action-sharing)
- course_chain：生活化活动展示 → 她询问 → 讲真实片段 → 转共同体验。
- supplement_chain：真实片段 → 让她参与 → 坏坏带入 → 轻邀约。

## <a id="ob-c025-002"></a>OB-C025-002：见面方式的窗口分级
- 标签：[窗口](../../课程场景标签.md#tag-concept-window)、[行动配合](../../课程场景标签.md#tag-signal-action-cooperation)、[邀约阶段](../../课程场景标签.md#tag-stage-invitation)
- course_chain：上门/外约/她附近 → 判断投入等级 → 匹配现场能力 → 决定行动。
- supplement_chain：高窗上门 → 中窗外约 → 低窗她附近 → 持续低投入筛选。

## <a id="ob-c025-003"></a>OB-C025-003：快速邀约后的真实性刹车
- 标签：[真实性测试](../../课程场景标签.md#tag-scene-sincerity-test)、[后撤](../../课程场景标签.md#tag-action-withdraw)、[推拉](../../课程场景标签.md#tag-action-push-pull)
- course_chain：快速“甜甜待一起” → 她质疑滥情 → 简短刹车 → 换真实生活内容。
- supplement_chain：简短否认 → 给观察权 → 坏坏反测 → 降速换题。

## <a id="ob-c025-004"></a>OB-C025-004：邀约后继续强化画面
- 标签：[未来画面](../../课程场景标签.md#tag-scene-future-picture)、[场景描绘](../../课程场景标签.md#tag-action-scene-painting)、[邀约落地](../../课程场景标签.md#tag-stage-invitation-landing)
- course_chain：答应见面 → 继续聊共同画面 → 建立见面预期 → 临近执行确认。
- supplement_chain：活动细节 → 感受画面 → 坏坏暧昧 → 执行保温。

## <a id="ob-c025-005"></a>OB-C025-005：把她的素材变成共同语言
- 标签：[分享](../../课程场景标签.md#tag-action-sharing)、[调侃](../../课程场景标签.md#tag-action-tease)、[特殊性](../../课程场景标签.md#tag-psych-specialness)

- supplement_chain：当场复用 → 追细节 → 坏坏专属 → 转共同体验。

## <a id="ob-c025-006"></a>OB-C025-006：夜店接人的执行窗口
- 标签：[位置确认](../../课程场景标签.md#tag-scene-location-confirm)、[行动配合](../../课程场景标签.md#tag-signal-action-cooperation)、[直接](../../课程场景标签.md#tag-action-direct)
