# C031 OB 双链

## <a id="ob-c031-001"></a>OB-C031-001：中性短回复不自动关窗
- 标签：[低承诺回复](../../课程场景标签.md#tag-signal-low-commitment)、[窗口](../../课程场景标签.md#tag-concept-window)、[直接](../../课程场景标签.md#tag-action-direct)
- course_chain：表情包/嗯呐 → 仍在听 → 寒暄后说正题 → 看真实反应。
- supplement_chain：直接正题 → 轻转场 → 状态确认 → 后撤观察。

## <a id="ob-c031-002"></a>OB-C031-002：上次消息没回的修复
- 标签：[真实性测试](../../课程场景标签.md#tag-scene-sincerity-test)、[行动证明](../../课程场景标签.md#tag-action-proof)、[补安全感](../../课程场景标签.md#tag-action-reassure)
- course_chain：她追责漏回 → 承认 → 修机照片验证 → 恢复可信度。
- supplement_chain：先承认 → 给事实 → 给特殊性 → 无证据时简洁。

## <a id="ob-c031-003"></a>OB-C031-003：聊天记录很重要
- 标签：[表达特殊性](../../课程场景标签.md#tag-action-specialness)、[暧昧](../../课程场景标签.md#tag-stage-flirting)、[浪漫](../../课程场景标签.md#tag-tone-romantic)
- course_chain：修机/备份 → 聊天记录承载关系 → `miss you` → 升温。
- supplement_chain：温柔 → 浪漫 → 坏坏特殊 → 克制升级。

## <a id="ob-c031-004"></a>OB-C031-004：没和别的小姐姐聊天
- 标签：[关系测试](../../课程场景标签.md#tag-scene-relationship-test)、[确认特殊性](../../课程场景标签.md#tag-psych-specialness)、[反测试](../../课程场景标签.md#tag-action-counter-test)
- course_chain：她阴阳确认 → 给当前偏爱 → 不把自己说成无人喜欢。
- supplement_chain：当前专注 → 特殊性 → 坏坏反问 → 阶段边界。

## <a id="ob-c031-005"></a>OB-C031-005：只白天见与今晚想见
- 标签：[时间顾虑](../../课程场景标签.md#tag-scene-time-concern)、[有兴趣但犹豫](../../课程场景标签.md#tag-psych-interested-hesitant)、[场景描绘](../../课程场景标签.md#tag-action-scene-painting)
- course_chain：她要求白天 → 他解释今晚想见不是只在夜里 → 描绘以后白天陪伴。
- supplement_chain：尊重白天 → 今晚短见 → 双时间线 → 坏坏推拉。

## <a id="ob-c031-006"></a>OB-C031-006：反复说你不在意
- 标签：[确认认真程度](../../课程场景标签.md#tag-psych-seriousness)、[推拉](../../课程场景标签.md#tag-action-push-pull)、[框架](../../课程场景标签.md#tag-concept-frame)
- course_chain：多轮态度确认 → 先真诚 → 重复时变硬 → 再软化并给行动。
- supplement_chain：第一次软接 → 重复时立态度 → 对抗后软化 → 行动收口。

## <a id="ob-c031-007"></a>OB-C031-007：接人要求与工作收尾
- 标签：[邀约落地](../../课程场景标签.md#tag-stage-invitation-landing)、[位置确认](../../课程场景标签.md#tag-scene-location-confirm)、[行动配合](../../课程场景标签.md#tag-signal-action-cooperation)
- course_chain：她要接 → 工作未完 → 解释与给时间 → 位置/接人执行。
- supplement_chain：立即去接 → 她先过来 → 折中汇合 → 做不到则改期。

全部 `pending_user_confirmation`；正式 OB：0。
