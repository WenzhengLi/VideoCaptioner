# C026 OB 双链

## <a id="ob-c026-001"></a>OB-C026-001：低信息但持续回复
- 标签：[持续回复](../../课程场景标签.md#tag-signal-continuous-reply)、[黄灯](../../课程场景标签.md#tag-signal-yellow)、[窗口](../../课程场景标签.md#tag-concept-window)
- course_chain：长期短回 → 不判死刑 → 低压持续 → 等位置/时间行动信号。
- supplement_chain：状态接续 → 二选一 → 坏坏轻挑 → 等行动窗口。

## <a id="ob-c026-002"></a>OB-C026-002：她主动让你接下班
- 标签：[窗口上升](../../课程场景标签.md#tag-signal-window-up)、[位置确认](../../课程场景标签.md#tag-scene-location-confirm)、[行动配合](../../课程场景标签.md#tag-signal-action-cooperation)
- course_chain：她给接下班窗口 → 拿时间位置 → 计算路程 → 接送落地。
- supplement_chain：直接确认 → 带计划 → 坏坏认领 → 给退路。

## <a id="ob-c026-003"></a>OB-C026-003：临出发下头的台阶
- 标签：[有兴趣但犹豫](../../课程场景标签.md#tag-psych-interested-hesitant)、[提供筹码](../../课程场景标签.md#tag-action-chips)、[邀约落地](../../课程场景标签.md#tag-stage-invitation-landing)
- course_chain：已约 → 她嫌收拾麻烦 → 降低准备成本 → 已出发增强执行 → 重新配合。
- supplement_chain：降成本 → 理解推进 → 坏坏带领 → 明确取消时改期。

## <a id="ob-c026-004"></a>OB-C026-004：见面后的关系索要
- 标签：[主动报备](../../课程场景标签.md#tag-signal-active-report)、[关系测试](../../课程场景标签.md#tag-scene-relationship-test)、[轻占有](../../课程场景标签.md#tag-action-light-possession)
- course_chain：见面后她报备 → 提情人节礼物 → 索要关系位置 → 再次见面窗口。
- supplement_chain：接关系信号 → 坏坏认领 → 行动补足 → 慢速观察。

## <a id="ob-c026-005"></a>OB-C026-005：强烈指责时认怂道歉
- 标签：[框架转换](../../课程场景标签.md#tag-action-frame-shift)、[后撤](../../课程场景标签.md#tag-action-withdraw)、[直接](../../课程场景标签.md#tag-action-direct)
- course_chain：欺骗/白嫖指责 → 不对抗 → 道歉 → 必要事实解释 → 停止刺激。
- supplement_chain：先道歉 → 澄清事实 → 停止刺激 → 允许修复或退出。

## <a id="ob-c026-006"></a>OB-C026-006：颜控与订婚决策
- 标签：[认真程度](../../课程场景标签.md#tag-psych-seriousness)、[关系测试](../../课程场景标签.md#tag-scene-relationship-test)、[直接](../../课程场景标签.md#tag-action-direct)
- course_chain：对脸犹豫 → 先继续相处 → 恋爱/婚姻分线 → 观察性格与协作后决策。
- supplement_chain：延长观察 → 看生活协作 → 区分吸引与适配 → 给决策边界。

全部 `pending_user_confirmation`；正式 OB：0。
