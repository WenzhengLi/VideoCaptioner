# C028 OB 双链

## <a id="ob-c028-001"></a>OB-C028-001：多条分享要逐项接住
- 标签：[分享](../../课程场景标签.md#tag-action-sharing)、[持续回复](../../课程场景标签.md#tag-signal-continuous-reply)

- supplement_chain：逐点回应 → 抓情绪主线 → 状态交换 → 坏坏奖励。

## <a id="ob-c028-002"></a>OB-C028-002：照片不要先负面定义
- 标签：[吸引建立](../../课程场景标签.md#tag-stage-attraction)、[分享](../../课程场景标签.md#tag-action-sharing)

- supplement_chain：中性展示 → 轻自信 → 坏坏索评 → 具体反馈。

## <a id="ob-c028-003"></a>OB-C028-003：前任问题中的关系态度
- 标签：[关系测试](../../课程场景标签.md#tag-scene-relationship-test)、[认真程度](../../课程场景标签.md#tag-psych-seriousness)、[价值观植入](../../课程场景标签.md#tag-action-value-injection)

- supplement_chain：简洁事实 → 成熟边界 → 关系标准 → 转当前。

## <a id="ob-c028-004"></a>OB-C028-004：软件认识不靠谱

## <a id="ob-c028-005"></a>OB-C028-005：胖和照片焦虑

## <a id="ob-c028-006"></a>OB-C028-006：太晚、上班与收拾顾虑
