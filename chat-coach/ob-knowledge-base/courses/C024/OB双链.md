# C024 OB 双链

## <a id="ob-c024-001"></a>OB-C024-001：过快推进引发目的怀疑
- 标签：[真实性测试](../../课程场景标签.md#tag-scene-sincerity-test)、[后撤](../../课程场景标签.md#tag-action-withdraw)、[邀约阶段](../../课程场景标签.md#tag-stage-invitation)

- supplement_chain：承认快 → 换公开场所 → 坏坏回收 → 给选择权。

## <a id="ob-c024-002"></a>OB-C024-002：高展示造成距离感
- 标签：[吸引建立](../../课程场景标签.md#tag-stage-attraction)、[需要安全感](../../课程场景标签.md#tag-psych-safety)、[分享](../../课程场景标签.md#tag-action-sharing)

- supplement_chain：真实日常 → 找共同点 → 坏坏拉近 → 行动验证。

## <a id="ob-c024-003"></a>OB-C024-003：已婚与花心核验
- 标签：[真实性测试](../../课程场景标签.md#tag-scene-sincerity-test)、[确认真实目的](../../课程场景标签.md#tag-psych-real-intent)、[直接](../../课程场景标签.md#tag-action-direct)

- supplement_chain：事实回答 → 点出顾虑 → 坏坏化解 → 行动验证。

## <a id="ob-c024-004"></a>OB-C024-004：具体计划代表被重视
- 标签：[邀约内容确认](../../课程场景标签.md#tag-scene-invite-content)、[邀约落地](../../课程场景标签.md#tag-stage-invitation-landing)、[提供筹码](../../课程场景标签.md#tag-action-chips)

- supplement_chain：承认疏漏 → 二选一 → 照顾式安排 → 坏坏带领。

## <a id="ob-c024-005"></a>OB-C024-005：第一次见面边界与对抗
- 标签：[关系测试](../../课程场景标签.md#tag-scene-relationship-test)、[推拉](../../课程场景标签.md#tag-action-push-pull)、[温柔](../../课程场景标签.md#tag-tone-gentle)

- supplement_chain：接受边界 → 保留吸引 → 坏坏推拉 → 照顾执行。

## <a id="ob-c024-006"></a>OB-C024-006：堵车爆胎后的重协调
