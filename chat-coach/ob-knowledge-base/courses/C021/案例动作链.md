1. 汕头案例：持续短回复 → 共同话题加热 → 多句软邀约试探 → 未接邀约后停止追问 → 兴趣与关心窗口 → 投入反转 → 问位置和行动信息 → 未来画面 → 咖啡邀约具体落地。

3. 报备案例：索要晚安 → 早安惯例 → 应酬关心转未来小酌 → 持续报备保温 → 接机与日期安排 → 计划变化后重协调 → 到达与见面执行 → 见面后继续维护。
