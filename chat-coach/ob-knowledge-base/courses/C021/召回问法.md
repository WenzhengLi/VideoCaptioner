# C021 召回问法

状态：`pending_user_confirmation`；以下只是候选召回入口，不是正式 Dify 配置。

- 她回得很短但一直回 / 只回可以什么意思 / 女生聊天不冷不热怎么办
- 她跳过我的邀约 / 不回答一起看剧是不是拒绝 / 软邀约没接怎么继续
- 女生问我几点睡 / 她关心我的作息是什么意思 / “你陪我吗”什么时候能说
- 女生连续主动找我 / 她开始报备了 / 女生投入变高后怎么回
- 女生在外面怎么问位置 / 她忙的时候还要聊天吗 / 怎么从位置判断能不能约
- 十小时没回怎么解释 / 聊得很热突然失联怎么补救 / 忙的时候怎么提前说
- 明明高窗口为什么还约不出来 / 女生很喜欢但没安全感 / 吸引之后聊什么
- 女生问前任什么意思 / 为什么不能详细讲前任 / 怎么简洁回答分手原因
- 她说缘分要持续 / 怎么从未来关系聊到具体见面 / 如何问休息时间
- 她说少喝点 / 怎么把喝酒聊成以后一起小酌 / 微醺场景怎么说自然
- 她要早安晚安 / 见面前怎么报备保温 / 约好后突然冷了怎么办
- 有空见怎么变成具体邀约 / 时间地点活动怎么确认 / 临时计划变了怎么改约

召回后必须结合关系阶段、回复速度、信息量、主动程度、是否明确拒绝、是否已有兴趣信号和是否已有具体行动判断，不能只凭关键词归类。
