# C023 OB 双链

## <a id="ob-c023-001"></a>OB-C023-001：独立人设下的表达欲
- 标签：[需要安全感](../../课程场景标签.md#tag-psych-safety)、[温柔](../../课程场景标签.md#tag-tone-gentle)、[分享](../../课程场景标签.md#tag-action-sharing)
- course_chain：强调独立 → 持续解释自己 → 先听完整 → 找孤独/被理解缺口。
- supplement_chain：接住 → 好奇缺口 → 坏坏拆壳 → 温柔占位。

## <a id="ob-c023-002"></a>OB-C023-002：追求者很多的门槛
- 标签：[框架](../../课程场景标签.md#tag-concept-frame)、[调侃](../../课程场景标签.md#tag-action-tease)、[特殊性](../../课程场景标签.md#tag-psych-specialness)
- course_chain：展示追求者 → 制造筛选门槛 → 不进入讨好竞争 → 转向她如何选择。
- supplement_chain：平静接住 → 反向筛选 → 坏坏调侃 → 温柔拉近。

## <a id="ob-c023-003"></a>OB-C023-003：进攻失败及时回收
- 标签：[后撤](../../课程场景标签.md#tag-action-withdraw)、[推拉](../../课程场景标签.md#tag-action-push-pull)、[黄灯](../../课程场景标签.md#tag-signal-yellow)
- course_chain：暧昧未接住 → 不连续强攻 → 回收换题 → 情绪恢复后再拉近。
- supplement_chain：自然收回 → 坏坏留钩 → 换到她的世界 → 低投入后撤。

## <a id="ob-c023-004"></a>OB-C023-004：筛选、嘴硬与退出辨别
- 标签：[窗口](../../课程场景标签.md#tag-concept-window)、[有兴趣但犹豫](../../课程场景标签.md#tag-psych-interested-hesitant)、[行动配合](../../课程场景标签.md#tag-signal-action-cooperation)
- course_chain：口头阻力 → 对照解释/主动/位置/行动 → 判断状态 → 选择推进或结束。
- supplement_chain：筛选接问 → 嘴硬读行动 → 真退出停止 → 不确定时验证。

## <a id="ob-c023-005"></a>OB-C023-005：到家后说要走
- 标签：[见面](../../课程场景标签.md#tag-stage-meeting)、[后撤](../../课程场景标签.md#tag-action-withdraw)、[需要安全感](../../课程场景标签.md#tag-psych-safety)
- course_chain：到家后说走 → 观察真实行动 → 给退出台阶 → 读取留下后的表达。
- supplement_chain：给自由 → 确认不适 → 轻松化解 → 情绪接住。

## <a id="ob-c023-006"></a>OB-C023-006：哭诉并索要拥抱
- 标签：[窗口上升](../../课程场景标签.md#tag-signal-window-up)、[温柔](../../课程场景标签.md#tag-tone-gentle)、[需要安全感](../../课程场景标签.md#tag-psych-safety)
- course_chain：强外壳松动 → 哭诉委屈 → 索要拥抱 → 先接情绪并对她好一点。
- supplement_chain：直接接住 → 温柔确认 → 坏坏缓和 → 次日稳定维护。

全部 `pending_user_confirmation`；正式 OB：0。
