# C027 OB 双链

## <a id="ob-c027-001"></a>OB-C027-001：共同旅行经历拉近

- supplement_chain：细节交换 → 相似感 → 坏坏拉近 → 场景邀约。

## <a id="ob-c027-002"></a>OB-C027-002：连续说不熟与先公开见
- 标签：[私密地点邀约](../../课程场景标签.md#tag-scene-private-invite)、[黄灯](../../课程场景标签.md#tag-signal-yellow)、[需要安全感](../../课程场景标签.md#tag-psych-safety)
- course_chain：私密邀约 → 连续理性顾虑 → 长回复仍投入 → 改变推进材料。
- supplement_chain：公开过渡 → 问具体顾虑 → 不丢框架回收 → 明确拒绝时后撤。

## <a id="ob-c027-003"></a>OB-C027-003：长回复但不松口
- 标签：[持续回复](../../课程场景标签.md#tag-signal-continuous-reply)、[窗口](../../课程场景标签.md#tag-concept-window)、[后撤](../../课程场景标签.md#tag-action-withdraw)
- course_chain：解释投入高 → 窗口未关但行动未同意 → 减速或侧移。
- supplement_chain：奖励投入 → 暂停争论 → 坏坏回收 → 观察行动。

## <a id="ob-c027-004"></a>OB-C027-004：感性冲不动转关系观
- 标签：[真实性测试](../../课程场景标签.md#tag-scene-sincerity-test)、[价值观植入](../../课程场景标签.md#tag-action-value-injection)、[框架](../../课程场景标签.md#tag-concept-frame)
- course_chain：暧昧强推无效 → 她要条件与可信度 → 补关系观和生活事实。
- supplement_chain：认识渠道 → 关系选择 → 具体生活 → 坏坏筛选。

## <a id="ob-c027-005"></a>OB-C027-005：减速但不丢方向
- 标签：[后撤](../../课程场景标签.md#tag-action-withdraw)、[框架](../../课程场景标签.md#tag-concept-frame)、[推拉](../../课程场景标签.md#tag-action-push-pull)
- course_chain：推进过快 → 退一步 → 保留见面意图 → 侧面补连接。
- supplement_chain：接受节奏 → 保留意图 → 坏坏推拉 → 换侧面。

## <a id="ob-c027-006"></a>OB-C027-006：东站接人执行
- 标签：[邀约落地](../../课程场景标签.md#tag-stage-invitation-landing)、[位置确认](../../课程场景标签.md#tag-scene-location-confirm)、[行动配合](../../课程场景标签.md#tag-signal-action-cooperation)

- supplement_chain：直接确认 → 降摩擦 → 坏坏接管 → 变更预案。

## <a id="ob-c027-007"></a>OB-C027-007：年龄嫌弃与表情包互怼
