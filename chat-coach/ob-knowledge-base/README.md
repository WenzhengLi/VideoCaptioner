# 课程 OB 正式知识库

本目录是聊天课程知识的正式组织层。`chat-coach/courses/` 保留课程扫描、原文核对和分析过程；只有经过用户确认的内容才能进入本目录的正式 OB 文件。

## 目录结构

```text
ob-knowledge-base/
├── README.md
├── 课程场景标签.md
├── OB总索引.md
├── _templates/
│   ├── 课程索引模板.md
│   ├── OB双链模板.md
│   ├── 补足候选模板.md
│   └── 召回问法模板.md
└── courses/
    ├── C001/
    │   └── 课程索引.md
    ├── ...
    └── C022/
        └── 课程索引.md
```

每课确认后逐步增加：

- `课程内容.md`：课程内容地图、概念和原始场景；
- `案例动作链.md`：课程实际案例的连续动作；
- `OB双链.md`：正式 `course_chain + supplement_chain` 对象；
- `补足与确认.md`：候选补足、用户修改和批准记录；
- `召回问法.md`：口语近义表达、用户问法和测试问题。

## 双向链接

### 标签 → OB

`课程场景标签.md` 中，每个标签维护“关联 OB”列表，链接到具体课程对象。

### OB → 标签

每个 `OB双链.md` 对象中的标签必须链接回 `课程场景标签.md` 的标签锚点。

这样可以同时完成：

```text
标签查场景 → 找到所有课程 OB
课程 OB 查标签 → 找到同类场景和相邻策略
```

## 内容状态

- `draft`：分析中；
- `pending_user_confirmation`：候选内容等待用户确认；
- `approved`：用户已确认，可进入正式知识库；
- `rejected`：用户否决，不进入正式知识库；
- `superseded`：已被新版对象替代。

## 建议的稳定标识

- 课程：`C019`；
- 标签：`TAG-SCENE-PRIVATE-INVITE`；
- 对象：`OB-C019-001`；
- 补足：`SUP-C019-001`；
- 测试问法：`QUERY-C019-001`。

文件名和标题可以调整，稳定 ID 一旦进入正式知识库后不随文案变化。
