# 课程场景标签

本文件是全局标签的唯一注册表。标签用于标记场景、信号、心理、动作、语气和课程概念，支持 OB 召回和跨课程关联；标签不能替代自然语言回答。

## 标签记录格式

每个标签包含：

- 稳定 ID；
- 标签键；
- 定义；
- 常见口语或近义表达；
- 相关标签；
- 关联 OB 反向链接。

## 关系阶段 `stage`

### <a id="tag-stage-break-ice"></a>TAG-STAGE-BREAK-ICE｜`stage:破冰`

- 定义：刚建立联系、重新开启或从陌生进入可连续互动。
- 相关标签：`scene:状态分享`、`scene:旧资源重启`、`action:分享`。
- 关联 OB：等待正式对象确认。

### <a id="tag-stage-attraction"></a>TAG-STAGE-ATTRACTION｜`stage:建立吸引`

- 定义：展示价值、制造男女感、特殊性和继续了解的兴趣。
- 相关标签：`action:框架转换`、`action:表达特殊性`。
- 关联 OB：等待正式对象确认。

### <a id="tag-stage-flirting"></a>TAG-STAGE-FLIRTING｜`stage:暧昧升温`

- 定义：双方已经出现明显男女互动、亲昵称呼、占有感或关系测试。
- 相关标签：`action:调侃`、`action:推拉`、`tone:暧昧`。
- 关联 OB：等待正式对象确认。

### <a id="tag-stage-invitation"></a>TAG-STAGE-INVITATION｜`stage:邀约`

- 定义：从聊天转入时间、地点、活动和见面安排。
- 相关标签：`action:邀约`、`scene:邀约内容确认`。
- 关联 OB：等待正式对象确认。

### <a id="tag-stage-invitation-landing"></a>TAG-STAGE-INVITATION-LANDING｜`stage:邀约落地`

- 定义：已经进入明确时间、地点、电话、位置和执行安排。
- 相关标签：`action:行动证明`、`signal:行动配合`。
- 关联 OB：等待正式对象确认。

### <a id="tag-stage-meeting"></a>TAG-STAGE-MEETING｜`stage:见面`

- 定义：线下接触、第一反应和现场互动。
- 相关标签：`scene:初次见面`、`action:现场决策`。
- 关联 OB：等待正式对象确认。

### <a id="tag-stage-escalation"></a>TAG-STAGE-ESCALATION｜`stage:关系升级`

- 定义：关系称呼、距离、情绪或亲密程度继续提高。
- 相关标签：`action:关系升级`、`signal:行动配合`。
- 关联 OB：等待正式对象确认。

## 场景 `scene`

### <a id="tag-scene-sharing"></a>TAG-SCENE-SHARING｜`scene:状态分享`

- 定义：分享刚醒、忙碌、工作、饮食、运动、情绪和日常状态。
- 口语：刚醒、在忙、今天好累、刚下班、在干嘛。
- 关联 OB：等待正式对象确认。

### <a id="tag-scene-old-lead-reopen"></a>TAG-SCENE-OLD-LEAD-REOPEN｜`scene:旧资源重启`

- 定义：聊天中断数日或更久后重新开启。
- 口语：最近忙什么、好久不见、失踪了、还记得我吗。
- 关联 OB：等待正式对象确认。

### <a id="tag-scene-fast-invite"></a>TAG-SCENE-FAST-INVITE｜`scene:快速邀约`

- 定义：认识时间较短时快速提出见面或共处。
- 相关标签：`scene:认识时间顾虑`、`signal:黄灯`。
- 关联 OB：等待正式对象确认。

### <a id="tag-scene-private-invite"></a>TAG-SCENE-PRIVATE-INVITE｜`scene:私密地点邀约`

- 定义：邀约涉及家里、酒店或其他私密共处空间。
- 口语：去你家吗、孤男寡女、在那里过夜吗。
- 关联 OB：等待正式对象确认。

### <a id="tag-scene-future-picture"></a>TAG-SCENE-FUTURE-PICTURE｜`scene:未来场景`

- 定义：让对方想象未来一起使用、吃饭、看电影、旅行或相处的画面。
- 相关标签：`action:场景描绘`、`concept:诱饵`。
- 关联 OB：等待正式对象确认。

### <a id="tag-scene-sincerity-test"></a>TAG-SCENE-SINCERITY-TEST｜`scene:真实性测试`

- 定义：确认喜欢是否真实、是否只是嘴上表达或只看外表。
- 口语：画饼、你只是看照片、男人都这样说、你是不是颜控。
- 关联 OB：等待正式对象确认。

### <a id="tag-scene-relationship-test"></a>TAG-SCENE-RELATIONSHIP-TEST｜`scene:关系确认测试`

- 定义：主动讨论是否确定关系、双方身份和认真程度。
- 口语：我们还没确定关系、你想跟我谈吗、我们算什么。
- 关联 OB：等待正式对象确认。

### <a id="tag-scene-invite-content"></a>TAG-SCENE-INVITE-CONTENT｜`scene:邀约内容确认`

- 定义：确认见面要做什么、活动价值和相处内容。
- 口语：去干嘛、做什么、为什么要过去。
- 关联 OB：等待正式对象确认。

### <a id="tag-scene-location-confirm"></a>TAG-SCENE-LOCATION-CONFIRM｜`scene:地点确认`

- 定义：确认真实见面地点及其含义。
- 口语：去哪里、是不是你家、这是酒店吗。
- 关联 OB：等待正式对象确认。

### <a id="tag-scene-money-test"></a>TAG-SCENE-MONEY-TEST｜`scene:金钱观测试`

- 定义：讨论花钱、付出、共同承担及爱与金钱的关系。
- 口语：给钱才是爱、你没有付出、你愿意为我花钱吗。
- 关联 OB：等待正式对象确认。

### <a id="tag-scene-vague-delay"></a>TAG-SCENE-VAGUE-DELAY｜`scene:模糊延期`

- 定义：对邀约不给具体替代时间，只说以后、有空或再说。
- 口语：等我有时间、以后再说、到时候看。
- 关联 OB：等待正式对象确认。

### <a id="tag-scene-competition"></a>TAG-SCENE-COMPETITION｜`scene:竞争者出现`

- 定义：相亲对象、其他追求者或异性信息进入聊天。
- 相关标签：`psychology:制造危机感`、`action:推拉`。
- 关联 OB：等待正式对象确认。

## 维护规则

1. 新标签必须先在本文件登记稳定 ID，课程 OB 才能使用；
2. 同义词不能随意创建多个标签，应写入同一标签的“口语”字段；
3. 每个正式 OB 必须反向链接到本文件中的标签锚点；
4. 每个标签必须维护关联 OB 链接，形成双向关系；
5. 删除标签前必须先迁移关联对象，稳定 ID 不重复使用；
6. 标签用于召回，不直接作为用户回答。

以下标签由本批课程原文中出现的场景增量注册；标签只用于召回，不直接作为客户答案。

- <a id="tag-action-value-injection"></a>**价值观植入**：用关系观、生活观补足判断条件。
- <a id="tag-concept-safety"></a>**安全感**：课程内关于可信、场所、接送与边界的总概念。
- <a id="tag-action-reassure"></a>**补安全感**：用认可、事实或行动降低顾虑。
- <a id="tag-concept-accelerator"></a>**踩油门**：关系推进的加速动作及速度判断。
- <a id="tag-concept-compliance-test"></a>**服从性测试**：课程将临时条件理解为对框架的观察。
- <a id="tag-action-choice"></a>**给选择**：提供两个以上真实可执行方案。
- <a id="tag-signal-active-question"></a>**主动追问**：女生主动催聊或继续抛问题。
- <a id="tag-stage-fast-attraction"></a>**快速建立吸引**：短时间内从破冰进入深聊或邀约。
- <a id="tag-scene-value-test"></a>**价值测试**：确认资源、能力、抢手程度或关系价值。
- <a id="tag-concept-high-availability"></a>**高位可得性**：有选择但对特定对象开放投入。
- <a id="tag-signal-long-message"></a>**长消息**：通过字数和解释投入判断互动仍在继续。
- <a id="tag-action-need-recognition"></a>**需求识别**：从表面问题读取情绪或关系需求。
- <a id="tag-scene-investment-test"></a>**付出测试**：询问礼物、时间、金钱或照顾标准。
- <a id="tag-stage-value-screening"></a>**价值筛选**：判断双方是否值得持续投入。
- <a id="tag-scene-first-meeting"></a>**首次见面**：线上印象转入线下重新评估。
- <a id="tag-psych-offline-reevaluation"></a>**线下重新评估**：线上关系意向不能替代真实见面判断。
- <a id="tag-action-live-decision"></a>**现场决策**：处理同事、行李、情绪等突发现场变量。
- <a id="tag-stage-icebreaking"></a>**破冰**：刚认识时建立连续体验和话题入口。
- <a id="tag-scene-reply-rhythm"></a>**回复节奏**：即时回复、漏回修复与停止连发。
- <a id="tag-scene-status-sharing"></a>**状态分享**：围绕当下正在做什么交换生活状态。
- <a id="tag-signal-time-investment"></a>**时间投入**：愿意安排、持续聊天或留出见面时间。
- <a id="tag-action-proof"></a>**行动证明**：用照片、位置、接送、准时等事实兑现说法。
- <a id="tag-scene-competitor"></a>**竞争者出现**：她主动展示追求者或其他异性资源。
- <a id="tag-action-counter-test"></a>**反测试**：不低位自证，以轻松反问或当前行为回应测试。
- <a id="tag-signal-low-commitment"></a>**低承诺回复**：表情包、嗯、哈哈等低信息但未中断的回复。
- <a id="tag-action-specialness"></a>**表达特殊性**：说明她为何获得不同于普通人的注意力。
- <a id="tag-scene-time-concern"></a>**时间顾虑**：太晚、上班、白天/晚上等安排阻力。
