# Knowledge Prompt v001

按 `P01` 至 `P06` 顺序执行。每课、每阶段使用独立 Cursor 命令和干净上下文；输入文件由命令
明确指定，输出到该课程对应目录。模型只能生成新文件，禁止修改输入文件。

每个阶段的共同规则：

1. 只做本阶段职责，不提前总结下一阶段内容。
2. 不确定内容必须保留，并写入 `uncertainties`。
3. 任何结论都通过 `evidence_spans` 指向时间戳、行号、PDF 页码或截图编号。
4. 区分观察事实、课程讲师观点、模型推断和人工确认。
5. 明确拒绝、报警、威胁、欺骗或施压内容必须进入 `safety_flags`。
6. 输出必须是可解析 JSON，不添加代码围栏或 JSON 之外的解释。
