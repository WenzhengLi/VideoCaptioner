# Knowledge Prompt v003

从 `knowledge-v002` 复制的并行版本；**不得覆盖**已有 v002 课程产物。

## 相对 v002 的变更重点

- P03：宽边界优先；同案讲解/课板穿插不得丢进 `unassigned`；未分配比例 >20% 必须自检并写入 `uncertainties`。
- 其余阶段继承 v002 的证据、讲师主张分层与安全边界规则。
- Compact P02 规则见 `prompts/knowledge-v003-compact/`。

## 使用方式

调度器通过 `--output-version knowledge-v003` 与 `-PromptRoot prompts\knowledge-v003`
（及紧凑复核目录）启用。历史课程保留 `knowledge-v002` 输出以便对比完整度与检索质量。
